## 📝 数据源说明

### 腾讯财经 (qt.gtimg.cn)
- **优点**: 稳定、快速
- **数据**: 上证指数、深证成指实时行情
- **更新**: 交易时间实时推送

### 新浪财经 (vip.stock.finance.sina.com.cn)
- **优点**: 数据全面
- **数据**: A 股个股行情、换手率
- **限制**: 板块数据偶尔不稳定

---

## 🔄 数据更新机制

1. **交易时间** (9:30-11:30, 13:00-15:00)
   - 每 3 秒自动刷新
   - 实时显示最新数据

2. **非交易时间**
   - 显示缓存数据
   - 每日 14:59 自动保存缓存

3. **缓存管理**
   - 位置：`cache/market-data.json`
   - 按日期存储
   - 自动清理过期缓存

## ⚠️ 注意事项

1. **网络环境**: 某些数据源在特定网络下可能不稳定
2. **API 限流**: 建议刷新间隔不低于 3 秒
3. **数据延迟**: 免费 API 可能有 1-3 秒延迟
4. **停牌股票**: 已自动过滤 ST、*ST 股票
