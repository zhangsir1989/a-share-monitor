## 问题根因

**输入**: "sh000001 上证指数"  
**错误**: "未找到该股票，请输入 6 位股票代码" ❌

**根本原因**: 后端 API 逻辑缺陷

**旧逻辑**:
```javascript
// 只匹配纯 6 位数字
if (/^\d{6}$/.test(query)) {
  // 直接查询
} else {
  // 进入搜索逻辑（但 stocks.js 中搜索的是 code 字段，不是完整输入）
}
```

**问题**: 输入"sh000001"不匹配 `^\d{6}$` 正则，进入搜索逻辑但找不到匹配

---

## 修复方案

### 修复后端 API 逻辑 ✅

**新逻辑**:
```javascript
// 提取代码（支持 sh/sz 前缀）
const codeMatch = query.match(/(sh|sz)?(\d{6})/);
if (codeMatch) {
  const code = codeMatch[1] ? codeMatch[0] : codeMatch[2];
  const result = await fetchStockDetail(code);
  res.json(result);
  return;
}

// 如果提取失败，再进入搜索逻辑
```

**支持格式**:
- ✅ `sh000001` - 带前缀
- ✅ `000001` - 纯代码
- ✅ `上证指数` - 中文名称
- ✅ `SHZS` - 拼音首字母

## 技术细节

### 正则表达式
```javascript
/(sh|sz)?(\d{6})/
```

**匹配结果**:
- 输入："sh000001" → `["sh000001", "sh", "000001"]`
- 输入："000001" → `["000001", undefined, "000001"]`

**处理逻辑**:
```javascript
const code = codeMatch[1] ? codeMatch[0] : codeMatch[2];
// 如果有前缀 (codeMatch[1])，返回完整代码 (codeMatch[0])
// 否则返回纯数字代码 (codeMatch[2])
```
