## 问题描述

**现象**: 输入"上证指数"点击查询，显示"未找到该股票"

**原因**: 
1. ❌ 股票数据中没有指数
2. ❌ 后端不支持带前缀的代码（如 sh000001）
3. ❌ 前端缓存问题

---

---

### 修复 2：支持带前缀的代码 ✅

**修复逻辑**:
```javascript
// 如果已经带市场前缀
if (code.startsWith('sh') || code.startsWith('sz')) {
  market = code.substring(0, 2);
  code = code.substring(2);
}
// 如果是 6 位数字，判断市场（只在没有 market 时）
else if (/^\d{6}$/.test(code)) {
  // ... 自动判断市场
}
```

**支持格式**:
- `sh000001` - 带前缀
- `000001` - 纯代码（自动判断）
- `上证指数` - 中文名称
