## 🚀 使用流程

### 1. 访问系统

打开浏览器访问：
- 本地：`http://localhost:3000`
- 局域网：`http://<你的 IP>:3000`

### 2. 登录界面

首次访问会自动跳转到登录页面，输入用户名和密码。

### 3. 记住我功能

- ✅ 勾选"记住我"：下次自动登录（凭证保存在本地浏览器）
- ⬜ 不勾选：每次访问需重新登录

### 4. 访问系统

登录成功后，可访问所有页面：
- 📊 首页：`http://localhost:3000`
- 📈 证券行情：`http://localhost:3000/stock`
- ⭐ 自选股：`http://localhost:3000/custom`

### 5. 退出登录

点击页面右上角的 🚪 按钮，确认后退出登录。

---

## 🛡️ 安全特性

### 会话管理
- 会话有效期：24 小时
- 自动过期：关闭浏览器后会话失效
- 安全存储：使用 httpOnly Cookie，防止 XSS 攻击

### 密码保护
- 密码验证：服务器端验证
- 错误提示：模糊错误信息，防止枚举攻击
- 登录日志：服务器记录登录尝试

### 记住我功能
- 本地存储：凭证加密存储在浏览器 localStorage
- 仅当前浏览器：不会跨设备同步
- 随时清除：取消勾选"记住我"并退出登录即可清除

## 🔧 API 接口

### 登录
```http
POST /api/login
Content-Type: application/json

{
  "username": "zhangsir",
  "password": "111111"
}
```

响应：
```json
{
  "success": true,
  "message": "登录成功"
}
```

### 登出
```http
GET /api/logout
```

响应：
```json
{
  "success": true,
  "message": "已退出登录"
}
```

### 检查登录状态
```http
GET /api/auth/status
```

响应：
```json
{
  "isAuthenticated": true,
  "username": "zhangsir"
}
```

## ⚙️ 配置选项

### 修改管理员账户

编辑 `src/server.js`，找到以下代码：

```javascript
const ADMIN_USER = {
  username: 'zhangsir',
  password: '111111'
};
```

修改为你自己的用户名和密码。

### 会话配置

在 `src/server.js` 中修改会话中间件：

```javascript
app.use(session({
  secret: 'your-secret-key',  // 生产环境使用固定密钥
  cookie: {
    maxAge: 24 * 60 * 60 * 1000  // 会话有效期（毫秒）
  }
}));
```

## 🐛 故障排除

### 问题：无法访问登录页面
**解决**：检查服务器是否启动，访问 `http://localhost:3000/login.html`

### 问题：登录后仍然被重定向到登录页
**解决**：清除浏览器缓存和 Cookie，重新登录

### 问题：记住我功能不生效
**解决**：检查浏览器是否允许 localStorage，确保勾选"记住我"

### 问题：登出后仍能访问页面
**解决**：清除浏览器缓存，关闭浏览器重新打开
