## 问题描述

### 问题 1：点击查询报错
**现象**: 输入"300014 亿纬锂能"后点击查询，提示"请输入 6 位股票代码"

**原因**: 输入框包含股票名称，API 无法解析

### 问题 2：中文名称无法查询
**现象**: 输入"招商证券"点击查询，提示"未找到该股票"

**原因**: 
1. 前端没有提取纯代码
2. 后端不支持中文名称搜索
3. stocks.js 缺少部分券商股票数据

---

## 修复方案

### 修复 1：前端提取股票代码

**新增函数**:
```javascript
// 从输入中提取股票代码
function extractCode(input) {
  if (!input) return '';
  // 匹配 6 位数字代码
  const match = input.match(/\d{6}/);
  return match ? match[0] : input.trim();
}
```

**应用位置**:
- 查询按钮点击事件
- 回车键事件
- 搜索建议选择

**示例**:
```
输入："300014 亿纬锂能"
提取："300014"
查询：✅ 成功
```

### 修复 2：后端支持中文搜索

**修改 API 逻辑**:
```javascript
app.get('/api/stock/:query', async (req, res) => {
  let query = req.params.query.trim();
  
  // 如果是 6 位数字，直接查询
  if (/^\d{6}$/.test(query)) {
    const result = await fetchStockDetail(query);
    res.json(result);
    return;
  }
  
  // 如果是中文或拼音，先搜索找到代码
  const searchResult = stocks.filter(stock => {
    if (stock.name.includes(query)) return true;
    if (stock.pinyin.includes(query.toUpperCase())) return true;
    return false;
  }).slice(0, 1);
  
  if (searchResult) {
    const result = await fetchStockDetail(searchResult.code);
    res.json(result);
  } else {
    res.json({ success: false, message: '未找到该股票' });
  }
});
```

## 注意事项

### 输入建议
1. **推荐使用下拉建议** - 点击搜索建议最准确
2. **代码查询最快** - 6 位数字直接查询
3. **中文可能重名** - 如"华泰证券"和"海通证券"拼音都是 HTZQ

- 目前支持 288 只股票和 ETF
- 主要是主流股票和热门 ETF
