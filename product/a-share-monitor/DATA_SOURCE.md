# 数据源说明

## 当前使用的数据源

### ✅ 已实现实时数据

| 数据模块 | 数据源 | 状态 | 说明 |
|---------|--------|------|------|
| 市场成交量 | 腾讯财经 | ✅ 实时 | 上证/深证成交额、成交量 |
| 高换手率股票 | 新浪财经 | ✅ 实时 | TOP50 股票，含换手率、价格 |

### ⚠️ 使用演示数据

| 数据模块 | 状态 | 说明 |
|---------|------|------|
| 涨停板块 | ⚠️ 演示 | 因 API 限制，当前使用按涨跌幅排序的板块数据 |
| 板块资金流 | ⚠️ 演示 | 因 API 限制，当前使用模拟数据 |

## 数据更新频率

- **交易时间**: 每 3 秒自动刷新（可配置 1-10 秒）
- **非交易时间**: 显示缓存数据
- **缓存时间**: 每日收盘后（14:59）自动保存

## API 接口

### 成交量数据
```bash
curl http://localhost:3000/api/market-volume
```

返回示例：
```json
{
  "totalVolume": 64877,
  "totalAmount": 64876.75,
  "shVolume": 30725,
  "szVolume": 34151,
  "shAmount": 3.07,
  "szAmount": 3.42,
  "shRatio": "47.36",
  "szRatio": "52.64"
}
```

### 高换手率股票
```bash
curl http://localhost:3000/api/high-turnover
```

### 涨停板块
```bash
curl http://localhost:3000/api/limit-up-sectors
```

### 板块资金流
```bash
curl http://localhost:3000/api/sector-cashflow
```

## 数据源切换

如需使用其他数据源，修改 `src/data-api.js` 中的对应函数。

支持的数据源：
- 腾讯财经 (qt.gtimg.cn)
- 新浪财经 (finance.sina.com.cn)
- 东方财富 (eastmoney.com) - 需要正确的网络环境

## 注意事项

1. **网络环境**: 某些数据源在特定网络环境下可能无法访问
2. **API 限流**: 建议刷新间隔不低于 3 秒
3. **数据延迟**: 免费 API 可能存在 1-3 秒延迟
4. **停牌股票**: 已自动过滤 ST、*ST 股票

## 扩展数据源

如需更准确的数据，可以考虑：
1. 购买付费数据服务（如 Tushare、JoinQuant）
2. 使用券商提供的 API
3. 本地部署数据服务
