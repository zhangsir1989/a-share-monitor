## 技术实现

### 价格生成逻辑

```javascript
// 分段生成价格
if (i < 30) {
  // 开盘 30 分钟：围绕开盘价波动
  basePrice = open;
  volatility = prevClose * 0.025;
} else if (i < 120) {
  // 上午：从开盘价向中间价过渡
  basePrice = open + (currentPrice - open) * progress * 0.4;
} else if (i < 130) {
  // 下午开盘：小幅波动
  basePrice = open + (currentPrice - open) * 0.4;
} else {
  // 下午：向收盘价靠拢
  basePrice = open + (currentPrice - open) * (0.4 + progress * 0.6);
}
```

### 成交量生成逻辑

```javascript
// 成交量分布
if (i < 30) {
  volumeFactor = 3 + Math.random() * 2;  // 开盘最大
} else if (i >= 30 && i < 60) {
  volumeFactor = 2 + Math.random() * 1;  // 开盘后较大
} else if (i >= 200) {
  volumeFactor = 2.5 + Math.random() * 2; // 尾盘最大
} else if (i >= 120 && i < 130) {
  volumeFactor = 1.5 + Math.random() * 0.5; // 下午开盘
} else {
  volumeFactor = 0.3 + Math.random() * 0.3; // 中间最小
}
```
