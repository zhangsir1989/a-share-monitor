# 🔍 股票搜索功能升级

## 更新时间
2026-03-31 09:40 UTC

---

## 新增功能

### 1. 模糊搜索
支持多种搜索方式：
- ✅ **股票代码** - 6 位代码（如：600519）
- ✅ **股票名称** - 中文全称（如：贵州茅台）
- ✅ **拼音首字母** - 拼音缩写（如：GZMT）

### 2. 实时搜索建议
- ✅ **输入时自动搜索** - 输入 1 个字符即开始匹配
- ✅ **下拉列表显示** - 最多显示 10 条匹配结果
- ✅ **键盘导航** - 支持上下箭头选择
- ✅ **回车确认** - 快速选择搜索

---

## 使用方式

### 方式 1：输入搜索
1. 在搜索框输入股票代码/名称/拼音
2. 自动显示匹配的股票列表
3. 点击或回车选择股票
4. 自动查询并显示行情

### 方式 2：键盘操作
- **↓ 下箭头** - 选择下一条建议
- **↑ 上箭头** - 选择上一条建议
- **Enter** - 确认选择并查询
- **Esc** - 关闭建议列表

### 方式 3：热门股票
点击底部热门股票快捷查询

## 技术实现

### 后端 API

**接口**: `/api/stock/search?q=关键词`

**响应**:
```json
{
  "success": true,
  "data": [
    {
      "code": "600519",
      "name": "贵州茅台",
      "pinyin": "GZMT",
      "market": "sh"
    }
  ],
  "count": 2
}
```

### 前端功能

**输入监听**:
- `input` 事件 - 触发搜索（300ms 防抖）
- `keydown` 事件 - 键盘导航
- `focus` 事件 - 聚焦时显示建议
- `click` 外部 - 关闭建议

**搜索逻辑**:
```javascript
// 模糊匹配代码、名称、拼音
const results = stockList.filter(stock => {
  if (stock.code.startsWith(query)) return true;
  if (stock.name.includes(query)) return true;
  if (stock.pinyin.includes(query)) return true;
  return false;
});
```
