## 功能概述

优化导航结构，添加**标签页切换**功能，将首页和证券行情整合到同一页面。

---

## 标签页结构

### 🏠 首页（默认）
显示核心监控功能：
- ✅ 成交量监控
- ✅ 涨停板块
- ✅ 板块资金流
- ✅ 高换手率 TOP50

### 📈 证券行情
显示个股查询功能：
- ✅ 股票搜索
- ✅ 行情详情
- ✅ 热门股票

## 技术实现

### HTML 结构
```html
<header class="header">
  <div class="header-top">
    <!-- 时间和标题 -->
  </div>
  <div class="header-controls">
    <!-- 标签页导航 -->
    <div class="tab-nav">
      <button class="tab-btn active" data-tab="home">🏠 首页</button>
      <button class="tab-btn" data-tab="stock">📈 证券行情</button>
    </div>
    <!-- 其他控件 -->
  </div>
</header>

<div class="tab-content">
  <div id="home-tab" class="tab-pane active">
    <!-- 首页内容 -->
  </div>
  <div id="stock-tab" class="tab-pane">
    <iframe src="/stock"></iframe>
  </div>
</div>
```

### CSS 样式
- **标签导航栏**: 深色背景，圆角设计
- **激活状态**: 蓝色高亮
- **内容切换**: 显示/隐藏对应标签页
- **证券行情**: 使用 iframe 嵌入

### JavaScript 逻辑
```javascript
function switchTab(tabName) {
  // 更新按钮状态
  // 更新内容显示
  // 加载 iframe（证券行情页）
}
```

## 使用方式

### 切换标签页
1. 点击顶部标签按钮
2. 内容区域自动切换
3. 证券行情页按需加载（首次点击时）

### URL 参数
可通过 URL 参数直接打开指定标签页：
```
http://localhost:3000/?tab=stock  # 打开证券行情页
http://localhost:3000/?tab=home   # 打开首页（默认）
```

## 优势

### 用户体验
- ✅ **一键切换** - 无需跳转页面
- ✅ **保持状态** - 刷新设置等保持不变

- ✅ **共享状态** - 数据源、刷新设置共享
- ✅ **按需加载** - 证券行情页延迟加载
