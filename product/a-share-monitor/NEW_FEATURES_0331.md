## 新增功能

### 1. 📈 分时走势图

**功能描述**:
- 显示股票当日每分钟的价格走势
- 包含成交量柱状图
- 显示昨收参考线
- 自动适配屏幕尺寸

**技术实现**:
- 使用 Canvas 绘制
- 240 个数据点（每分钟一个）
- 支持响应式布局

**界面位置**: 证券行情页面，股票基本信息下方

---

### 2. 🔗 相关可转债

**功能描述**:
- 显示股票对应的所有可转债
- 实时显示转债价格和涨跌幅
- 点击转债可快速查询
- 无转债时自动隐藏区域

**技术实现**:
- 自动匹配正股代码
- 实时获取转债行情
- 网格布局展示

**界面位置**: 分时图下方

## 使用说明

### 查看分时图

1. **搜索股票** - 输入股票代码或名称
2. **自动显示** - 查询成功后自动加载分时图
3. **图表说明**:
   - 蓝色线：价格走势
   - 黄色虚线：昨收价
   - 红绿柱：成交量（红涨绿跌）

### 查看可转债

1. **自动显示** - 如果股票有对应转债，自动显示
2. **点击查询** - 点击转债卡片可快速查询转债详情
3. **信息展示**:
   - 转债代码
   - 转债名称
   - 当前价格
   - 涨跌幅

## API 接口

### 分时图 API

**请求**:
```http
GET /api/intraday/:code
```

**响应**:
```json
{
  "success": true,
  "data": [
    {
      "time": "09:30:00",
      "price": 100.50,
      "volume": 5000
    },
    ...
  ]
}
```

### 可转债 API

**请求**:
```http
GET /api/convertibles/:stockCode
```

**响应**:
```json
{
  "success": true,
  "data": [
    {
      "code": "110059",
      "name": "浦发转债",
      "price": 100.50,
      "change": 0.50,
      "changePercent": 0.50
    }
  ]
}
```
