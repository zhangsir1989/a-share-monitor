## 图表特点

### 1. 双坐标轴设计 ✅

**左侧坐标**: 绝对价格
- 显示 7 个价格级别
- 颜色根据涨跌显示（红涨绿跌）
- 字体加粗，清晰易读

**右侧坐标**: 涨跌幅百分比
- 显示 7 个百分比级别（±10.44%）
- 颜色根据涨跌显示（红涨绿跌）
- 与左侧价格一一对应

### 2. 价格线设计 ✅

**蓝色价格线**:
- 颜色：#108cee（东方财富蓝）
- 线宽：1.2px
- 走势清晰

**黄色昨收线**:
- 颜色：#f0a30a（黄色）
- 线宽：1.5px
- 作为 0% 基准参考

**浅蓝色填充**:
- 价格线下方渐变填充
- 从 10% 透明度渐变到 2%
- 增强视觉效果

### 3. 成交量图设计 ✅

**红绿柱状图**:
- 红色：价格上涨时（#ff4d4f）
- 绿色：价格下跌时（#52c41a）
- 柱宽自适应
- 高度反映成交量大小

**成交量标签**:
- 左侧显示 4 个级别
- 单位：万手
- 灰色字体

### 4. 网格线设计 ✅

**横向网格**:
- 7 条水平虚线
- 颜色：#e0e0e0（浅灰）
- 虚线样式：[3, 3]

**纵向网格**:
- 5 条垂直虚线
- 时间标记：9:30, 10:30, 11:30/13:00, 14:00, 15:00
- 分隔价格图和成交量图

## 技术实现

### 图表布局

```javascript
// 图表高度分配
const priceChartHeight = totalHeight * 0.65;  // 价格图 65%
const volumeChartHeight = totalHeight * 0.35; // 成交量图 35%
const volumeTop = priceTop + priceChartHeight + 10; // 10px 间隔
```

### 坐标计算

```javascript
// 价格坐标（以昨收为中心，上下各 10.44%）
const maxPrice = prevClose * 1.1044;
const minPrice = prevClose * 0.8956;
const priceRange = maxPrice - minPrice;

// Y 坐标计算
const y = padding.top + priceChartHeight * (maxPrice - price) / priceRange;
```

### 颜色逻辑

```javascript
// 价格颜色
const priceColor = price > prevClose ? '#ff4d4f' : 
                   (price < prevClose ? '#52c41a' : '#8b949e');

// 成交量颜色
const volumeColor = changePercent >= 0 ? '#ff4d4f' : '#52c41a';
```
