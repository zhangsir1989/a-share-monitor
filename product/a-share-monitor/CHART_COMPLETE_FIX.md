### 3. 成交量颜色修复 ✅

**问题**: 只显示绿色柱子，没有红色  
**原因**: 之前的逻辑是相对于昨收价判断涨跌  
**修复**: **改为相对于前一个时间点判断涨跌**

**旧逻辑**（错误）:
```javascript
// 相对于昨收价
const isUp = point.price >= prevClose;
```

**新逻辑**（正确）:
```javascript
// 相对于前一个时间点
if (index === 0) {
  isUp = point.price >= prevClose; // 第一个点相对于昨收
} else {
  isUp = point.price >= data[index - 1].price; // 其他点相对于前一个点
}
```

**修复效果**:
```
成交量颜色（新逻辑）:
  红色柱（涨）: 131 个 ← 现在有红色了！
  绿色柱（跌）: 109 个
  总计：240 个
```

**为什么现在有红有绿**:
- 虽然今天整体大跌（-10%）
- 但在下跌过程中有反弹
- 反弹时价格相对前一个点上涨 → 显示红色柱子
- 下跌时价格相对前一个点下跌 → 显示绿色柱子
