## 新增功能

### 1. 📅 北京时间显示

**位置**：页面左上角，标题下方

**显示内容**：
- 日期（年月日 + 星期）
- 时间（时：分：秒，实时跳动）
- 时区标识（北京时间）

**示例**：
```
2026 年 3 月 31 日 星期二  13:41:57  北京时间
```

**技术实现**：
- 每秒自动更新
- 自动转换为 UTC+8 北京时间
- 无论服务器在哪个时区，都显示北京时间

---

### 2. 📊 数据来源显示

**位置**：页面顶部，刷新频率左侧

**显示内容**：
- 成交量数据源
- 换手率数据源

**数据源标识**：
| 标识 | 颜色 | 说明 |
|------|------|------|
| **新浪** | 🟢 绿色 | 新浪财经 API（换手率准确） |
| **腾讯** | 🔵 蓝色 | 腾讯财经 API（稳定可靠） |
| **--** | ⚪ 灰色 | 未知/加载中 |

**示例**：
```
数据来源：成交量 [腾讯🔵]  换手率 [新浪🟢]
```

**技术实现**：
- 后端自动检测使用的数据源
- 前端每 3 秒刷新显示状态
- 优先使用新浪 API，失败自动切换腾讯

## 技术细节

### 前端修改

**HTML** (`public/index.html`):
- 添加 `.header-left` 容器
- 添加 `.datetime-display` 时间显示
- 添加 `.data-source` 数据源显示

**CSS** (`public/css/style.css`):
- 时间显示样式（等宽字体、颜色）
- 数据源标签样式（颜色编码）
- 响应式布局优化

**JavaScript** (`public/js/app.js`):
- `updateDateTime()` - 每秒更新北京时间
- `updateDataSources()` - 更新数据源标签
- 新增 `state.dataSources` 状态管理

### 后端修改

**API** (`src/data-api.js`):
- 添加 `dataSourceStatus` 状态追踪
- 添加 `getDataSourceStatus()` 接口
- 自动更新数据源状态

**服务器** (`src/server.js`):
- 新增 `/api/data-sources` 接口
- 返回当前使用的数据源

## 数据源说明

### 成交量数据
- **当前数据源**：腾讯财经
- **原因**：腾讯财经指数 API 稳定可靠
- **更新频率**：每次请求实时获取

### 换手率数据
- **优先**：新浪财经（真实换手率）
- **备用**：腾讯财经（估算换手率）
- **切换逻辑**：新浪失败自动用腾讯
