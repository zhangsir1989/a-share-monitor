### 坐标轴说明

**右侧坐标**:
- **红色数字**: 涨跌幅百分比（+10% 到 -10%）
- **灰色数字**: 对应的绝对价格
- **黄色粗线**: 0% 基准线（昨收价）

**下方坐标**:
- **时间**: 09:30, 10:30, 11:30, 13:00, 14:00, 15:00
- **网格**: 虚线分隔

**成交量**:
- **红色柱**: 价格上涨时的成交量
- **绿色柱**: 价格下跌时的成交量

## 技术实现

### 数据生成逻辑

```javascript
// 1. 计算涨跌幅百分比
const changePercent = ((price - prevClose) / prevClose * 100);

// 2. 确定坐标范围
const maxChange = Math.max(...changePercents, 2);  // 至少±2%
const minChange = Math.min(...changePercents, -2);

// 3. 绘制百分比网格
for (let i = -10; i <= 10; i += 2) {
  const y = padding.top + priceHeight * (maxChange - i) / changeRange;
  // 绘制网格线和标签
}

// 4. 绘制成交量
const barHeight = (volume / maxVolume) * volumeHeight * 0.9;
const color = changePercent >= 0 ? 'red' : 'green';
```
