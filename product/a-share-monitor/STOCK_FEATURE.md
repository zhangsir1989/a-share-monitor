# 📈 证券行情查询功能说明

## 更新时间
2026-03-31 09:10 UTC

---

## 功能概述

新增**证券行情查询页面**，支持输入股票代码查询详细的个股行情信息。

---

## 访问方式

### 1. 直接访问
```
http://localhost:3000/stock
```

### 2. 从首页导航
点击首页顶部的 **"📈 证券行情"** 按钮

### 3. 带参数访问
```
http://localhost:3000/stock?code=600519
```
自动查询贵州茅台

---

## 功能特性

### 🔍 股票查询
- **支持输入**：
  - 6 位股票代码（如：600519）
  - 股票名称（如：贵州茅台）- 待实现
  - 拼音首字母（如：GZMT）- 待实现

- **自动识别市场**：
  - 6/9 开头 → 沪市（sh）
  - 0/3 开头 → 深市（sz）

### 📊 行情数据

显示 14 项核心行情指标：

| 数据项 | 说明 | 单位 |
|--------|------|------|
| **今开** | 今日开盘价 | 元 |
| **最高** | 今日最高价 | 元 |
| **涨停** | 涨停价格 | 元 |
| **换手** | 换手率 | % |
| **成交量** | 成交手数 | 万手 |
| **市盈 (动)** | 动态市盈率 | - |
| **总市值** | 总市值 | 万亿/亿 |
| **昨收** | 昨日收盘价 | 元 |
| **最低** | 今日最低价 | 元 |
| **跌停** | 跌停价格 | 元 |
| **量比** | 量比指标 | - |
| **成交额** | 成交金额 | 亿 |
| **市净** | 市净率 | - |
| **流通市值** | 流通市值 | 万亿/亿 |

## 使用说明

### 查询股票
1. 在搜索框输入股票代码（6 位数字）
2. 点击"🔍 查询"按钮或按回车键
3. 页面显示详细行情信息

### 快捷查询
1. 点击底部"热门股票"中的任意股票
2. 自动填充并查询该股票

### 清空重置
1. 点击"✕ 清空"按钮
2. 清空搜索框和结果显示

---

## 数据更新

- **交易时间**: 实时数据（延迟 1-3 秒）
- **非交易时间**: 显示最新收盘数据
- **自动刷新**: 需手动刷新页面

---

## 注意事项

1. **输入格式**: 目前仅支持 6 位股票代码查询
2. **名称搜索**: 股票名称搜索功能待实现
3. **数据延迟**: 免费 API 可能有 1-3 秒延迟
4. **休市时间**: 非交易时间显示收盘数据
