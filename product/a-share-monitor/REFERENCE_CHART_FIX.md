## 设计细节

### 1. 坐标轴设计 ✅

**左侧价格**:
- 位置：虚线左边
- 颜色：红涨绿跌
  - 上涨：#ff5252（红色）
  - 下跌：#4caf50（绿色）
  - 昨收：#90a4ae（灰色）
- 对齐：右对齐，紧贴虚线

**右侧百分比**:
- 位置：虚线右边
- 颜色：与左侧价格对应
- 对齐：左对齐

### 2. 网格线设计 ✅

**横向网格**:
- 只保留 0 轴（昨收价）的虚线
- 颜色：#f0a30a（黄色）
- 样式：虚线 [4, 4]

**纵向网格**:
- 5 条垂直虚线（9:30, 10:30, 11:30/13:00, 14:00, 15:00）
- 颜色：#606060（深灰）
- 贯穿价格图和成交量图

### 3. 价格线设计 ✅

**蓝色价格线**:
- 颜色：#2979ff（亮蓝色）
- 线宽：1px（细线）
- 走势清晰

**浅蓝色填充**:
- 从价格线到 0 轴（昨收价）
- 渐变：15% → 2% 透明度

### 4. 成交量设计 ✅

**柱子样式**:
- 宽度：细柱子（70% 宽度）
- 颜色：全部绿色 #4caf50
- 高度：反映成交量大小

**金额标签**:
- 位置：左侧
- 显示：总成交额（万元）
- 级别：3 级（0, 50%, 100%）

## 技术实现

### 图表布局

```javascript
// 图表高度分配
const priceChartHeight = totalHeight * 0.60;  // 价格图 60%
const volumeChartHeight = totalHeight * 0.40; // 成交量图 40%
const volumeTop = priceTop + priceChartHeight + 15; // 15px 间隔
```

### 网格线逻辑

```javascript
// 只绘制 0 轴虚线
const zeroY = padding.top + priceChartHeight * (maxPrice - prevClose) / priceRange;
ctx.strokeStyle = '#f0a30a';
ctx.setLineDash([4, 4]);
ctx.beginPath();
ctx.moveTo(padding.left, zeroY);
ctx.lineTo(width - padding.right, zeroY);
ctx.stroke();
```

### 成交量柱子

```javascript
// 细柱子（70% 宽度）
const barWidth = Math.max(1, (width - padding.left - padding.right) / data.length * 0.7);

// 全部绿色
ctx.fillStyle = '#4caf50';
ctx.fillRect(x - barWidth/2, volumeTop + volumeChartHeight - barHeight, barWidth, barHeight);
```
