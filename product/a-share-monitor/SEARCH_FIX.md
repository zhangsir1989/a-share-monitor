## 问题描述

搜索时出现大量重复股票，例如搜索"YWLN"显示 5 条相同的"300014 亿纬锂能"。

**原因**: `stocks.js` 文件中存在 31 条重复的股票记录。

---

## 修复方案

### 1. 数据去重
使用 Node.js 脚本处理股票列表：
```javascript
// 按代码去重
const seen = new Set();
const uniqueStocks = [];
for (const stock of stocks) {
  if (!seen.has(stock.code)) {
    seen.add(stock.code);
    uniqueStocks.push(stock);
  }
}
```

### 2. 结果统计
- **原始数量**: 237 只
- **去重后**: 206 只
- **移除重复**: 31 只
