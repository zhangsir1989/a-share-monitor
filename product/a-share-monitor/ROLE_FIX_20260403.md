# 🔧 角色判断逻辑修复

**修复时间：** 2026-04-03 20:40 UTC  
**问题：** 管理员登录后看到的是普通用户菜单（修改密码），而不是管理员菜单（用户管理）

---

## 🔍 问题原因

**根本原因：角色值不匹配**

- **后端返回：** `role: '1'`（管理员）或 `role: '0'`（普通用户）
- **localStorage 存储：** `'1'` 或 `'0'`
- **前端判断：** `if (userRole === 'admin')` ❌

**问题代码：**
```javascript
// 登录时保存（正确）
localStorage.setItem('userRole', result.role || 'user');  // 保存 '1' 或 '0'

// 判断时使用（错误）
const userRole = localStorage.getItem('userRole') || 'user';
if (userRole === 'admin') {  // ❌ 永远不会匹配！
  // 显示管理员菜单
}
```

---

## ✅ 修复内容

**修改所有前端 JS 文件中的角色判断逻辑：**

| 文件 | 修改前 | 修改后 |
|------|--------|--------|
| `public/js/app.js` | `userRole === 'admin'` | `userRole === '1'` |
| `public/js/stock.js` | `userRole === 'admin'` | `userRole === '1'` |
| `public/js/custom.js` | `userRole === 'admin'` | `userRole === '1'` |
| `public/js/change-password.js` | `userRole === 'admin'` | `userRole === '1'` |
| `public/js/users.js` | ✅ 已正确 | ✅ 无需修改 |

**修复后的代码：**
```javascript
// 根据用户角色显示不同菜单
const userRole = localStorage.getItem('userRole') || '0';
console.log('📋 当前用户角色:', userRole, '(1=管理员，0=普通用户)');

const usersMenuLink = document.querySelector('.sidebar-link[data-page="users"]');
const changePasswordLink = document.querySelector('.sidebar-link[data-page="change-password"]');

// 管理员显示用户管理菜单（role='1'）
if (usersMenuLink) {
  if (userRole === '1') {  // ✅ 正确判断
    usersMenuLink.style.display = 'flex';
    console.log('✅ 显示用户管理菜单');
  } else {
    usersMenuLink.style.display = 'none';
    console.log('🚫 隐藏用户管理菜单（非管理员）');
  }
}

// 普通用户显示修改密码菜单（role='0'）
if (changePasswordLink) {
  if (userRole !== '1') {  // ✅ 正确判断
    changePasswordLink.style.display = 'flex';
    console.log('✅ 显示修改密码菜单');
  } else {
    changePasswordLink.style.display = 'none';
    console.log('🚫 隐藏修改密码菜单（管理员）');
  }
}
```
