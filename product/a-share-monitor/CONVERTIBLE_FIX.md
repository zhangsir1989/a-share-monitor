## 使用说明

### 如何查看可转债

1. **搜索股票** - 输入股票代码或名称
2. **自动显示** - 如果该股票有转债，自动显示在"相关可转债"区域
3. **点击查询** - 点击转债卡片可快速查询转债行情

### 示例

**查询浦发银行**:
```
搜索：600000 或 浦发银行
显示：
  🔗 相关可转债
  ┌──────────────┐
  │ 110059       │
  │ 浦发转债     │
  │ 100.50 +0.5% │
  └──────────────┘
```

**查询中国平安**:
```
搜索：601318 或 中国平安
显示：
  （无转债区域，自动隐藏）
```
