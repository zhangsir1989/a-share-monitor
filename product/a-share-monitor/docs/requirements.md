# A 股实时监控系统 - 需求文档

## 📋 系统概述

**系统名称：** A 股实时监控系统（A-Share Monitor System）

**功能定位：** 实时监控 A 股市场数据，提供可视化监控界面

**目标用户：** 股票投资者、交易员

---

## 🎯 核心功能

### 1. 首页监控看板

#### 1.1 成交量卡片
- 总成交额（亿元/万亿元）
- 总成交量（万手）
- 沪市成交额及占比
- 深市成交额及占比
- 数据源：腾讯财经 API

#### 1.2 涨停板块 TOP10
- 板块名称
- 涨停家数
- 板块涨跌幅
- 按平均涨幅排序

#### 1.3 板块资金流 TOP20
- 板块名称
- 主力净流入（亿元）
- 净流入率

#### 1.4 高换手率 TOP50
- 股票代码/名称
- 现价/涨跌幅
- 换手率/实际换手率
- 成交额
- 所属行业

### 2. 证券行情查询

#### 2.1 股票搜索
- 支持股票代码（6 位数字）
- 支持股票名称（中文）
- 支持拼音首字母
- 搜索建议下拉框

#### 2.2 行情展示
- 基本信息：代码、名称、类型
- 当前价格、涨跌额、涨跌幅
- 开盘/最高/最低/昨收
- 涨停/跌停价格
- 成交量/成交额
- 换手率/量比
- 市盈率/市净率
- 总市值/流通市值

#### 2.3 分时图
- 价格曲线（蓝色平滑曲线）
- 昨收线（黄色虚线）
- 成交量柱状图（红涨绿跌）
- 时间轴标签（9:30/10:30/11:30/13:00/14:00/15:00）
- 价格标签（左侧）和百分比标签（右侧）
- 动态 Y 轴范围（基于实际高低点）

#### 2.4 可转债关联
- 自动显示相关可转债
- 可转债价格和涨跌幅

### 3. 数据刷新

#### 3.1 自动刷新
- 默认刷新间隔：3 秒
- 可配置间隔：1/2/3/5/10 秒
- 暂停/继续功能
- 手动刷新按钮

#### 3.2 市场状态判断
- 未开盘（9:15 前）：清空数据
- 集合竞价（9:15-9:25）：显示数据
- 交易中（9:30-11:30, 13:00-15:00）：实时更新
- 已收盘（15:00 后）：显示最后数据
- 周末/节假日：休市

## 📡 数据源

### 实时行情
| 数据源 | 用途 | 优先级 |
|--------|------|--------|
| 腾讯财经 API | 个股行情、成交量 | 主 |
| 新浪财经 API | 高换手率数据 | 备用 |

### API 接口
- `http://qt.gtimg.cn/q={code}` - 个股行情
- `http://smartbox.gtimg.cn/s3/` - 搜索建议
- `http://vip.stock.finance.sina.com.cn/` - 新浪数据

---

## 🌐 API 接口

### 系统 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/all` | GET | 获取全部市场数据 |
| `/api/market-volume` | GET | 成交量数据 |
| `/api/limit-up-sectors` | GET | 涨停板块 |
| `/api/high-turnover` | GET | 高换手率 |
| `/api/sector-cashflow` | GET | 板块资金流 |
| `/api/data-sources` | GET | 数据源状态 |
| `/api/stock/:query` | GET | 个股详情 |
| `/api/stock/search` | GET | 搜索建议 |
| `/api/intraday/:code` | GET | 分时图数据 |
| `/api/convertibles/:code` | GET | 可转债 |

### 页面路由

| 路由 | 说明 |
|------|------|
| `/` | 首页监控看板 |
| `/stock` | 证券行情查询 |
