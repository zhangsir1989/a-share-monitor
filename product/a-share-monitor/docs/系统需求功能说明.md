## 一、系统概述

### 1.1 项目背景

随着中国资本市场的不断发展，A 股投资者对实时行情工具的需求日益增长。为帮助投资者快速掌握市场动态、捕捉热点板块、提高投资决策效率，特开发 A 股实时监控系统。

本系统通过对接多家财经数据源，实时监控 A 股市场数据，为投资者提供多维度、可视化的市场监控功能，支持盘中实时监控和盘后分析。

### 1.2 系统定位

| 项目 | 说明 |
|------|------|
| **目标用户** | A 股投资者、证券从业人员、金融分析师、量化交易人员 |
| **使用场景** | 交易时间实时监控、盘后复盘分析、热点板块追踪、资金流向监控、个股行情查询、指数行情查询 |
| **核心价值** | 实时数据、多维度监控、直观展示、快速响应、数据安全 |
| **部署方式** | B/S 架构，支持 PC/平板/手机访问 |
| **运行环境** | Windows/Linux/Mac 操作系统，Node.js v16+ |

### 1.3 技术架构

#### 1.3.1 整体架构

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   前端      │────▶│   后端 API   │────▶│   数据源    │
│ (HTML/CSS/  │     │ (Node.js +   │     │ (腾讯财经/  │
│  JS)        │     │  Express)    │     │  新浪财经)  │
└─────────────┘     └──────────────┘     └─────────────┘
       │                    │
       ▼                    ▼
┌─────────────┐     ┌──────────────┐
│  浏览器缓存 │     │  本地缓存    │
│ (localStorage)│    │  (JSON 文件)  │
└─────────────┘     └──────────────┘
```

#### 1.3.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端** | HTML5 + CSS3 + JavaScript | 原生开发，无需框架 |
| **后端** | Node.js + Express | 轻量级 Web 服务 |
| **数据源** | 腾讯财经、新浪财经 | 权威稳定数据源 |
| **存储** | 本地 JSON 文件 + localStorage | 轻量级数据缓存 |
