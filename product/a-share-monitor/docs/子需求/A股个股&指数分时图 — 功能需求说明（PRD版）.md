# A股个股&指数分时图 — 功能需求说明（PRD版）

# 1. 功能概述

实现A股市场**个股**与**市场指数**的当日分时走势图展示功能，基于实时行情数据绘制日内价格/点位曲线、成交量柱状图，并展示核心行情指标，用于用户观察日内走势、强弱及成交活跃度。

# 2. 适用范围

- A股个股（主板、创业板、科创板、ST等）

- A股市场指数（上证指数、深证成指、创业板指、沪深300等宽基/行业指数）

# 3. 界面结构

整体分为上下两部分：

1. **上方：分时走势主图**

2. **下方：分时成交量副图**

# 4. 主图绘制逻辑

## 4.1 坐标轴

- **X轴：交易时间**

    - 固定交易日时间轴：09:30 – 11:30，13:00 – 15:00

    - 中间11:30–13:00为休市间隔，曲线不绘制。

- **Y轴：价格/指数点位**

    - 个股：以**昨日收盘价**为中心，自适应上下浮动区间

    - 指数：以**昨日收盘点位**为中心，自适应展示

## 4.2 曲线绘制

1. **分时现价线**

    - 数据源：逐笔/快照最新价

    - 表现：连续折线，实时追加最新点

2. **均价线（仅个股显示，指数不显示）**$$均价 = \frac{当日累计成交金额}{当日累计成交量}$$

    - 计算公式：

    - 指数无均价线，不绘制

3. **昨日收盘基准线**

    - 水平虚线贯穿全图

    - 作为当日涨跌判断基准

4. **涨跌停参考线（仅个股）**

    - 根据个股涨跌幅限制绘制±10%/±20%/±5%水平虚线

    - 指数无涨跌停，不绘制

## 4.3 颜色规则

- 现价线在昨收上方：红色系

- 现价线在昨收下方：绿色系

- 昨收线：灰色/淡蓝色虚线

# 5. 成交量副图逻辑

## 5.1 数据粒度

按**分钟级成交量**绘制柱状图。

## 5.2 颜色规则（A股通用）

- 当分钟**最新价 ≥ 上一分钟收盘价**：红柱

- 当分钟**最新价 ＜ 上一分钟收盘价**：绿柱

## 5.3 展示内容

- 分钟成交量柱

- 当日累计成交量/成交额（文字展示）

# 6. 行情指标展示规则

## 6.1 个股分时图需展示

- 最新价

- 涨跌额、涨跌幅

- 开盘价、最高价、最低价

- 均价

- 当日成交量、成交额

- 换手率、量比（可选）

## 6.2 指数分时图需展示

- 当前点位

- 涨跌额、涨跌幅

- 开盘点位、最高点位、最低点位

- 成交量、成交额

- **不展示**：均价、换手率、量比、盘口

# 7. 数据更新逻辑

- 实时模式：通过行情推送/轮询获取最新快照

- 每收到新一分钟数据：

    - 追加现价点，刷新折线

    - 追加一根成交量柱

    - 更新右侧/顶部行情数值

- 15:00 收盘后：停止更新，保持最终形态

# 8. 个股与指数差异说明（开发必看）

|项|个股分时|指数分时|
|---|---|---|
|均价线|显示|不显示|
|涨跌停线|显示|不显示|
|换手率/量比|显示|不显示|
|盘口五档|可选显示|不支持|
|Y轴范围|按涨跌停限制|自由自适应|
# 9. 接口字段要求（极简版）

如需对接接口，最少需要以下字段：

- 标的代码、名称、类型（个股/指数）

- 昨收价/昨收点位

- 最新价/点位

- 分钟时间戳

- 分钟成交量

- 当日累计成交量、累计成交金额

- 开盘、最高、最低

- 涨跌幅限制（个股专用）
