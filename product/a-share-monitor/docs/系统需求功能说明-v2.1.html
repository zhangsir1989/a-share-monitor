一、系统概述

1.1 项目背景

随着中国资本市场的不断发展，A 股投资者对实时行情工具的需求日益增长。为帮助投资者快速掌握市场动态、捕捉热点板块、提高投资决策效率，特开发 A 股实时监控系统。

本系统通过对接多家财经数据源，实时监控 A 股市场数据，为投资者提供多维度、可视化的市场监控功能，支持盘中实时监控和盘后分析。

1.2 系统定位

项目说明
目标用户A 股投资者、证券从业人员、金融分析师、量化交易人员
使用场景交易时间实时监控、盘后复盘分析、热点板块追踪、资金流向监控、个股行情查询
核心价值实时数据、多维度监控、直观展示、快速响应、数据安全
部署方式B/S 架构，支持 PC/平板/手机访问
运行环境Windows/Linux/Mac 操作系统，Node.js v16+

1.3 技术架构

层级技术选型说明
前端HTML5 + CSS3 + JavaScript原生开发，无需框架
后端Node.js + Express轻量级 Web 服务
数据源腾讯财经、新浪财经权威稳定数据源
存储本地 JSON 文件轻量级数据缓存
运行环境Node.js v16+跨平台支持
进程管理PM2后台运行、自动重启

二、功能需求

2.1 全市场成交量监控

功能描述

实时监控全市场成交量和成交额变化，帮助投资者判断市场活跃度和资金动向。

功能要求

  • ✅ 实时展示 A 股当前总成交量和总成交额
  • ✅ 沪市、深市分开统计显示
  • ✅ 沪市、深市占比百分比显示
  • ✅ 数据实时刷新（按页面设置频率）
  • ✅ 成交额单位自动适配（亿元/万亿元）

数据字段

字段说明单位状态
总成交量全市场成交手数总和万手✅ 已实现
总成交额全市场成交金额总和亿元/万亿元✅ 已实现
沪市成交量上海证券交易所成交量万手✅ 已实现
沪市成交额上海证券交易所成交额亿元✅ 已实现
深市成交量深圳证券交易所成交量万手✅ 已实现
深市成交额深圳证券交易所成交额亿元✅ 已实现
沪市占比沪市成交量占总成交量比例%✅ 已实现
深市占比深市成交量占总成交量比例%✅ 已实现

2.2 涨停板块监控

功能描述

实时统计各板块上涨股票数量，展示市场热点板块，帮助投资者快速发现市场主线。

功能要求

  • ✅ 实时展示上涨股票数前 10 的板块
  • ✅ 显示板块名称、上涨家数、板块涨幅
  • ✅ 板块按平均涨幅排序
  • ✅ 显示板块成分股数量
  • ⚠️ 板块图/色块展示热度（未实现）

完成度：85%

2.3 高换手率 TOP50 榜

功能描述

实时展示全市场 A 股实际换手率前 50 的股票，帮助投资者发现高活跃度和资金关注度高的标的。

功能要求

  • ✅ 实时展示全市场 A 股实际换手率前 50 的股票
  • ✅ 支持按成交额、成交量、涨幅、换手率等排序
  • ✅ 自动剔除 ST、*ST 股票
  • ✅ 自动剔除停牌股票

完成度：95%

2.4 板块资金流

功能描述

实时监控板块资金流入流出情况，识别主力资金动向，辅助投资者判断板块热度。

功能要求

  • ✅ 实时展示板块资金流入流出情况
  • ✅ 自动识别主力资金流入/流出板块
  • ✅ 展示板块资金流前 20 的板块
  • ✅ 显示主力净流入金额和净流入率
  • ✅ 显示大单、小单净流入

完成度：90%

四、部署说明

4.1 系统要求

项目要求
操作系统Windows 10/11、Linux、macOS
Node.jsv16 或更高版本（推荐 v18+）
内存最低 512MB，推荐 1GB
磁盘空间最低 100MB
网络需要能够访问腾讯财经、新浪财经 API
浏览器Chrome/Firefox/Edge 等现代浏览器

4.2 部署步骤

Windows 系统

  1. 安装 Node.js：https://nodejs.org/ 下载 LTS 版本
  2. 解压项目到目录（如 D:\a-share-monitor\）
  3. 双击 start.bat 启动
  4. 浏览器访问 http://localhost:3000

Linux 系统

# 1. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 2. 进入项目目录
cd a-share-monitor

# 3. 安装依赖
npm install --registry=https://registry.npmmirror.com

# 4. 启动服务
npm start

# 5. 访问 http://localhost:3000

macOS 系统

# 1. 安装 Node.js (使用 Homebrew)
brew install node

# 2. 进入项目目录并安装依赖
cd a-share-monitor && npm install

# 3. 启动服务
npm start

# 4. 访问 http://localhost:3000
