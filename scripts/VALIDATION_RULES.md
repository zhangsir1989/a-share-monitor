## ✅ 校验规则

### 规则 1：Script 标签闭合 (script-closed)
**要求**：所有 `<script>` 标签必须正确闭合

| 状态 | 示例 |
|------|------|
| ❌ 错误 | `<script src="app.js">` |
| ✅ 正确 | `<script src="app.js"></script>` |

### 规则 2：Link 标签闭合 (link-closed)
**要求**：`<link>` 标签应使用自闭合语法

| 状态 | 示例 |
|------|------|
| ❌ 错误 | `<link rel="stylesheet" href="style.css">` (未自闭合) |
| ✅ 正确 | `<link rel="stylesheet" href="style.css">` |

### 规则 3：Meta 标签闭合 (meta-closed)
**要求**：`<meta>` 标签应使用自闭合语法

| 状态 | 示例 |
|------|------|
| ❌ 错误 | `<meta charset="UTF-8">` (未自闭合) |
| ✅ 正确 | `<meta charset="UTF-8">` |

## 📝 工作流程

### 修改 HTML 文件后

```bash
# 1. 修改 HTML 文件
vim public/index.html

# 2. 运行自动校验
./scripts/auto-validate.sh

# 3. 如果校验通过
✅ 提交代码

# 4. 如果校验失败
❌ 修复错误 → 重新校验 → 通过后提交
```

---

## 🔧 常见问题

### 问题 1：Script 标签未闭合
**症状**：页面空白，控制台无任何输出

**修复**：
```html
<!-- 错误 -->
<script src="js/app.js"

<!-- 正确 -->
<script src="js/app.js"></script>
```

### 问题 2：版本号未更新
**症状**：浏览器缓存旧文件

**修复**：
```html
<!-- 更新版本号强制刷新 -->
<script src="js/app.js?v=202604011830"></script>
```
