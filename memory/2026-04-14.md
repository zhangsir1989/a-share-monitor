## 2026-04-14 01:30 UTC - 删除分组时同时删除该分组下的所有股票

### 用户需求

**用户反馈**：删除分组时，希望该分组下的所有股票都被删除，而不是"回归默认自选股"。

**示例**：
- AI 算力分组有 1 只股票
- 删除 AI 算力分组时，该股票应该被删除
- 之前的逻辑：股票 type 改为 1（回归默认自选股）❌
- 期望逻辑：直接删除股票 ✅

### 修改内容

#### 1. 后端 API 修改 (src/group-api.js)

**修改前**：
```javascript
// 将该分组的股票 type 改回 1（回归默认自选股）
db.run(`UPDATE custom_stocks SET type = 1 WHERE user_id = '${userId}' AND type = ${groupType}`);
```

**修改后**：
```javascript
// 查询该分组下的股票数量
const stockCount = db.exec(`SELECT COUNT(*) FROM custom_stocks WHERE user_id = '${userId}' AND type = ${groupType}`);
const count = stockCount.length > 0 ? stockCount[0].values[0][0] : 0;

if (count > 0) {
  // 删除该分组下的所有股票
  db.run(`DELETE FROM custom_stocks WHERE user_id = '${userId}' AND type = ${groupType}`);
  console.log(`✅ 已删除 ${count} 只股票`);
}

// 删除分组
db.run(`DELETE FROM custom_groups WHERE id = ${id} AND user_id = '${userId}'`);
```

**返回消息**：
```javascript
res.json({ success: true, message: `分组删除成功，已删除 ${count} 只股票` });
```

#### 2. 前端确认提示更新 (public/js/group-manager.js)

**修改前**：
```javascript
if (!confirm('确定要删除这个分组吗？分组内的股票将回归默认自选股。')) {
  return false;
}
```

**修改后**：
```javascript
if (!confirm('⚠️ 确定要删除这个分组吗？\n\n删除后：\n- 该分组下的所有股票将被删除\n- 此操作不可恢复')) {
  return false;
}
```

**更新位置**：
- `deleteGroup(groupId)` 函数
- `deleteGroupByType(groupType)` 函数
