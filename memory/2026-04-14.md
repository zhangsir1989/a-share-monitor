## 2026-04-14 07:00 UTC - 新增分时成交明细功能

### 用户需求

**原始需求**：
- 删除个股详细页逐笔成交明细功能
- 新增分时成交明细功能
- 实时获取分笔成交明细，3 秒刷新一次
- 独立模块，不影响其他逻辑

### 实现方案

#### 1. 新建前端模块 `trade-detail.js`

**文件**: `/root/.openclaw/workspace/product/a-share-monitor/public/stock-detail/js/trade-detail.js`

**模块结构**：
```javascript
const TradeDetail = {
  config: {
    refreshInterval: 3000,  // 3 秒刷新
    maxRecords: 100,        // 最多显示 100 条
    excludeCallAuction: true
  },
  state: {
    timer: null,
    isRunning: false,
    lastUpdate: null,
    trades: []
  },
  init(code, market),
  startAutoRefresh(),
  loadTrades(),
  render(),
  destroy()
};
```

**核心功能**：
- 自动定时刷新（3 秒/次）
- 从后端 API 获取成交数据
- 渲染成交列表（时间、价格、成交量、成交额、方向、订单类型）
- 方向标识：🟢买入、🔴卖出、⚪中性
- 订单类型标签：超（超大单）、大（大单）、中（中单）

#### 5. 后端 API

**接口**: `GET /api/stock/:code/tick-trades?limit=100`

**返回格式**：
```json
{
  "success": true,
  "data": [
    {
      "date": "2026-04-13",
      "time": "14:56:48",
      "volume": 53200,
      "price": 28.24,
      "amount": 150.2368,
      "direction": "sell",
      "orderType": "medium"
    }
  ]
}
```

**字段说明**：
- `direction`: 'buy' | 'sell' | 'neutral'
- `orderType`: 'superLarge' | 'large' | 'medium' | 'small'
- `amount`: 成交额（万元）
- `volume`: 成交量（股）
