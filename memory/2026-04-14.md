## 2026-04-14 10:30 UTC - 分时成交明细数据时效修复

### 用户反馈

**问题**: 成交明细显示的是昨天数据（2026-04-13），不是今天的实时行情

**需求**: 对接实时行情 API，显示今天的实时成交明细

### 问题分析

1. **MyData API 限制**: 
   - `/hsrl/zbjy/` 接口返回的是**交易日当天**的逐笔成交数据
   - 非交易时间或收盘后，数据不会更新
   - 当前日期 2026-04-14 是交易日，但 API 返回 2026-04-13 数据

2. **免费实时 API 限制**:
   - 腾讯财经：提供实时行情，但不提供完整逐笔成交
   - 东方财富：API 需要特定 referer 和参数，免费访问受限
   - 新浪财经：返回 Forbidden
   - **结论**: 免费实时逐笔成交 API 很难获取

### 解决方案

**方案**: 继续使用 MyData API + 前端明确标注数据时效

1. **后端修改** (`src/server.js`):
   - 使用 MyData 当天逐笔交易 API
   - 返回完整逐笔数据（按时间倒序）
   - 支持分页查询

2. **前端修改** (`public/stock-detail/index.html`):
   - 添加提示：`📊 数据截至昨日收盘 | 🔄 3 秒自动刷新`
   - 让用户了解数据是 T-1 的
   - 版本号更新为 v=2026041420

## 2026-04-14 11:00 UTC - 分时成交明细使用混合数据源（实时+T-1）

### 用户需求

**问题**: 当前使用的是 MyData 当天逐笔 API（/hsrl/zbjy/），每天 21:00 更新，非交易时间显示的是 T-1 数据

### 解决方案

**混合数据源方案**:

| 时间段（北京时间） | 数据源 | API | 更新频率 |
|------------------|--------|-----|---------|
| 9:30-11:30 | MyData 实时 | /hsrl/ssjy/ | 每分钟 |
| 13:00-15:00 | MyData 实时 | /hsrl/ssjy/ | 每分钟 |
| 其他时间 | MyData 逐笔 | /hsrl/zbjy/ | 21:00 |

### 实现细节

**后端逻辑** (`src/server.js`):

```javascript
// 检查当前时间（北京时间 UTC+8）
const now = new Date();
const beijingHour = (now.getUTCHours() + 8) % 24;
const currentTime = beijingHour * 60 + beijingMinutes;

// 交易时间：9:30-11:30, 13:00-15:00
const isTradingTime = (currentTime >= 570 && currentTime <= 690) || 
                      (currentTime >= 780 && currentTime <= 900);

if (isTradingTime) {
  // 使用实时 API
  const realtimeUrl = `https://api.mairuiapi.com/hsrl/ssjy/${codeNum}/${LICENCE}`;
  dataSource = 'MyData（实时）';
} else {
  // 使用逐笔 API
  const tradesUrl = `https://api.mairuiapi.com/hsrl/zbjy/${codeNum}/${LICENCE}`;
  dataSource = 'MyData（T-1）';
}
```
