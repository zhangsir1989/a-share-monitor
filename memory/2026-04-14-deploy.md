### 系统说明

**系统类型**: A 股监控系统（股票数据监控和管理平台）

**AI 模型**: 无 - 本系统不使用 AI 模型，主要功能为：
- 实时股票行情监控
- 股票数据展示（分时图、K 线图）
- 自选股管理
- 逐笔成交明细
- 多用户支持

### 数据库表结构

| 表名 | 说明 | 数据量 |
|------|------|--------|
| `users` | 用户账户管理 | 2 用户 |
| `custom_stocks` | 用户自选股 | 33 条 |
| `custom_groups` | 自选股分组 | 5 分组 |
| `securities` | 证券基础信息 | 需同步 |
| `tick_trade` | 逐笔成交数据 | 动态存储 |
| `scheduled_tasks` | 定时任务调度 | 0 任务 |

### 数据类型说明

#### 1. 用户数据 (users)
- user_id, username, password, role, is_active, created_at, last_login
- 默认管理员：admin / admin123

#### 2. 自选股分组 (custom_groups)
- **type 字段**: 分组类型标识
  - `type=0`: 持仓
  - `type=1`: 我的自选股（默认分组）
  - `type=2+`: 用户自定义分组
- 索引：(user_id, type) 复合索引

#### 3. 自选股 (custom_stocks)
- type 字段关联分组（一对一）
- UNIQUE(user_id, stock_code, stock_market, type)

#### 4. 证券信息 (securities)
- 类别：沪深 A 股、基金 (ETF)、可转债、沪深指数
- 市场：sh/sz/bj

#### 5. 逐笔成交 (tick_trade)
- 索引：(stock_code, trade_date)

### 部署脚本

#### scripts/init-database.js
**用途**: 初始化数据库表结构和默认数据

**功能**:
- 创建 6 个数据库表
- 创建索引优化查询
- 插入默认管理员账户

**使用**: `node scripts/init-database.js`

#### scripts/sync-securities.js
**用途**: 从 MyData API 同步证券信息

**功能**:
- 同步沪深 A 股、ETF、可转债、指数
- 自动解析市场前缀
- 避免 API 限流

**使用**: `node scripts/sync-securities.js`

#### scripts/export-data.js
**用途**: 导出数据为 JSON（备份）

**输出**:
- data/export/users.json
- data/export/custom_stocks.json
- data/export/custom_groups.json
- data/export/securities.json
- data/export/scheduled_tasks.json

**使用**: `node scripts/export-data.js`

#### scripts/import-data.js
**用途**: 从 JSON 导入数据（恢复）

**功能**:
- 读取 JSON 文件导入数据库
- 智能处理重复数据
- 支持增量导入

**使用**: `node scripts/import-data.js`

### 快速部署流程

```bash
# 1. 安装依赖
npm install

# 2. 初始化数据库
node scripts/init-database.js

# 3. 同步证券信息（可选）

# 4. 启动服务器
node src/server.js

# 5. 访问系统
# http://localhost:3000
# 登录：admin / admin123
```

### 数据备份与恢复
