## 自选股/持仓页顶部布局优化 (2026-04-11 08:17 UTC)

### 用户反馈
优化自选股页面上方布局（截图显示布局松散）

### 问题分析
1. 标题和日期时间显示分散
2. 控制按钮（返回首页、刷新间隔、暂停、刷新）布局混乱
3. 缺少状态反馈（运行中/已暂停）
4. 按钮样式不统一

### 优化方案

**1. Header 布局重构**
- 添加 `page-title-group` 容器：标题 + 日期时间
- 添加 `refresh-control-group` 容器：刷新间隔选择器 + 状态显示
- 布局更紧凑，视觉更整洁

**2. 刷新控制优化**
- 刷新间隔选择器：统一样式，带边框和背景
- 状态显示：运行中（绿色）/ 已暂停（灰色）
- 暂停按钮：显示"⏸️ 暂停" / "▶️ 继续"
- 立即刷新按钮：渐变紫色背景，突出显示

**3. 按钮样式**
- 暂停按钮：灰色背景 `var(--bg-card)`，悬停变深
- 刷新按钮：渐变紫色 `linear-gradient(135deg, var(--accent-color), var(--accent-hover))`
- 首页按钮：简化为"🏠 首页"

**4. JS 功能增强**
- `custom.js` 和 `positions.js` 同步更新
- 暂停时更新状态显示：
  - `refreshStatus.textContent = '已暂停'`
  - `refreshStatus.classList.add('paused')`
- 运行时恢复状态显示

**5. 响应式适配**
- 小屏幕（<1024px）：缩小标题字体和间距
- 超小屏幕（<768px）：隐藏"刷新间隔"标签

### 文件修改
- `custom.html` / `positions.html`：HTML 结构优化
- `custom.css`：新增 header 优化样式（~150 行）
- `custom.js` / `positions.js`：状态显示逻辑
