**MyData API 优先策略**:
```javascript
// 使用 MyData API 获取上证指数和深证成指的最新数据
const [shResp, szResp] = await Promise.all([
  axios.get(`https://api.mairuiapi.com/hsindex/latest/000001.SH/d/${MYDATA_LICENCE}`),
  axios.get(`https://api.mairuiapi.com/hsindex/latest/399001.SZ/d/${MYDATA_LICENCE}`)
]);

const shVolume = shData.v || 0;  // 成交量（股）
const szVolume = szData.v || 0;
const shAmount = shData.a || 0;  // 成交额（元）
const szAmount = szData.a || 0;

const totalAmount = (shAmount + szAmount) / 100000000;  // 转换为亿元
const totalVolume = (shVolume + szVolume) / 100000000;  // 转换为亿手
```

**返回数据格式**:
- `totalVolume`: 沪深总成交量（亿手）
- `totalAmount`: 沪深总成交额（亿元）
- `shVolume`: 沪市成交量（亿手）
- `szVolume`: 深市成交量（亿手）
- `shAmount`: 沪市成交额（亿元）
- `szAmount`: 深市成交额（亿元）
- `shRatio`: 沪市占比（%）
- `szRatio`: 深市占比（%）

**回退策略**: MyData API 失败时使用腾讯 API
