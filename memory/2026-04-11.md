## 资金诊断页面 + 净利润数据修复 (2026-04-11 08:03 UTC)

### 用户反馈
1. 分时走势页面右侧空白区域需要填充
2. 本年净利润数据没有值
3. 资金分析增加资金诊断按钮（靠右）
4. 点击资金诊断按钮跳转到新页面
5. 主力净流入值域颜色修改：正数红色，负数绿色

### 实现方案

**1. 资金诊断页面 (capital-diagnosis.html)**
- 综合评分（0-100 分，根据资金流向计算）
- 主力资金诊断（净流入、流入、流出、强度）
- 特大单诊断
- 大单诊断
- 资金流向趋势图（Canvas 柱状图）
- 诊断建议生成（根据评分和资金流向）
- 评分等级：优秀 (≥80)、良好 (≥60)、一般 (≥40)、较差 (<40)

**2. 个股详情页资金诊断按钮**
- 位置：资金分析卡片右上角（header-actions）
- 样式：渐变紫色背景，悬停效果
- 跳转：`/capital-diagnosis.html?code={股票代码}`

**3. 主力净流入颜色修复**
- CSS 修改：`.capital-value.net.inflow` = 红色，`.capital-value.net.outflow` = 绿色
- JS 逻辑：`mainNetflow >= 0 ? 'inflow' : 'outflow'`
- 效果：正数红色，负数绿色

**4. 净利润数据修复**
- 问题：只获取一条财务数据，本年净利润显示 '--'
- 修复：获取多条数据，第一条为本年，第二条为上年
- 回退机制：如果只有一条数据，单独请求上年数据
- 测试结果 (000001 平安银行)：
  - 本年净利润：426.33 亿元 ✅
  - 上年净利润：383.39 亿元 ✅

**5. 路由配置**
- 新增：`GET /capital-diagnosis` (需要登录)
- 文件：`public/stock-detail/capital-diagnosis.html`
