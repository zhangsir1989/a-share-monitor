
## 新增净利润字段 + 分时图右侧涨跌幅%

### 功能需求
1. 基础数据增加"上一年净利润"和"本年净利润"字段
2. 分时图右边黑色框填满，且右边增加%展示

### 实现方案

**1. 财务数据获取 (data-api.js)**
- 调用 MyData API 利润表接口：`/hsstock/financial/income/{code}.{market}/{licence}`
- 获取最新一期财务数据
- 解析 `jlr` 字段（净利润，单位：元）
- 转换为亿元显示：`lastYearNetProfit = jlr / 100000000`

**2. HTML 新增字段 (index.html)**
- 上年净利润：`<span id="basic-last-year-profit">`
- 本年净利润：`<span id="basic-current-year-profit">`

**3. UI 更新 (ui.js)**
- `updateBasicData()` 函数添加净利润字段显示
- 格式：`XX.XX 亿`

**4. 分时图右侧涨跌幅% (intraday-chart.js)**
- `drawGrid()` 函数右侧添加涨跌幅百分比标签
- `padding.right` 从 10px 增加到 70px
- 涨跌幅颜色：红涨绿跌
- 填充区域不透明度从 15% 提升到 30%

### 测试验证
- 000001 平安银行：上年净利润 426.33 亿元 ✅
- 分时图右侧显示涨跌幅%刻度 ✅
- 填充区域颜色加深 ✅
