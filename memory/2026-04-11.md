## 二级侧边栏折叠后左侧空白修复 (2026-04-11 09:32 UTC)

### 用户反馈
点击二级菜单后，左侧有一片空白（参考用户管理页面，不会出现此问题）

### 问题根因
**核心问题**: 二级侧边栏折叠时使用 `transform: translateX(-200px)` 实现位移动画
- 侧边栏视觉上是向左隐藏了
- 但 `left` 位置保持不变（240px）
- `container` 的 `padding-left` 仍然是 460px（包含二级侧边栏宽度）
- 导致左侧留下 200px 空白区域

**对比用户管理页面**:
- 用户管理页面没有二级侧边栏
- 使用 `container.no-sub-sidebar` 类，`padding-left: 260px`
- 所以不会出现空白问题

### 修复方案

**方案 1（原方案 - 有问题）**:
```css
.sub-sidebar.collapsed {
  width: 0;
  transform: translateX(-200px);  /* ❌ 问题：位移但 left 不变 */
}
```

**方案 2（修复方案）**:
```css
.sub-sidebar.collapsed {
  width: 0;
  overflow: hidden;
  transform: none;  /* ✅ 移除位移 */
  left: 240px;      /* ✅ 保持原位 */
}
```

**媒体查询增强**:
```css
@media (min-width: 769px) {
  /* 二级侧边栏折叠时，隐藏但保持位置 */
  .sub-sidebar.collapsed {
    width: 0 !important;
    overflow: hidden !important;
    left: 240px !important;
  }
  
  /* 主侧边栏折叠 + 二级侧边栏折叠 */
  .sidebar.collapsed .sub-sidebar.collapsed,
  .sidebar.collapsed + .sub-sidebar.collapsed {
    left: 60px !important;
  }
}
```
