
## 界面布局优化 + 成交量颜色修复

### 问题反馈
1. 好多数据没有显示
2. 每行展示 4 列数据
3. 分时图右侧空白区域需要填充
4. 成交量柱状图颜色有问题

### 修复方案

**1. 基本数据网格布局 (main.css)**
- 从 3 列改为 4 列：`grid-template-columns: repeat(4, 1fr)`
- gap 从 15px 调整为 12px，更紧凑

**2. 分时图统计信息优化 (intraday-chart.js)**
- `drawStats()` 函数重构
- 第一行：开、高、低、收（均匀分布）
- 第二行：涨跌、涨幅、成交量、成交额、振幅
- 充分利用顶部空间，消除右侧空白

**3. 成交量颜色修复 (intraday-chart.js)**
- 从半透明 `rgba(255, 77, 79, 0.7)` 改为实心颜色 `#ff4d4f`
- 红涨绿跌，颜色更明显

**4. 数据字段说明**
- 外盘、内盘：MyData API 暂无此数据（显示 '--'）
- 本年净利润：需等年报公布（显示 '--'）
- 其他字段均正常显示

### 效果
- 基本数据每行 4 列，布局更紧凑
- 分时图顶部统计信息完整，无空白区域
- 成交量柱状图颜色明显（红涨绿跌）
