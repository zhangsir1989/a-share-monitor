
## A 股监控系统修复记录

### 问题1：script 标签未闭合
**症状**：页面完全空白，控制台无任何日志
**原因**：`<script src="js/app.js?v=xxx"` 缺少 `</script>` 闭合标签
**修复**：确保所有 script 标签正确闭合

### 问题2：DOM 元素初始化时机
**症状**：控制台报错 `Cannot read property 'addEventListener' of undefined`
**原因**：`elements` 对象在文件顶部定义时 DOM 还未加载，所有元素都是 null
**修复**：将 `elements` 初始化移到 `init()` 函数内，确保 DOMContentLoaded 后执行

### 问题3：事件监听器注册时机
**症状**：`elements.refreshSelect is undefined`
**原因**：事件监听器在 `init()` 之前直接执行
**修复**：将事件监听器包装在 `initEventListeners()` 函数中，在 `init()` 内调用

### 关键修复代码
```javascript
// 错误写法（文件顶部）
const elements = {
  refreshSelect: document.getElementById('refresh-select'),  // null!
};

// 正确写法（init 函数内）
function init() {
  elements = {
    refreshSelect: document.getElementById('refresh-select'),
    // ...
  };
  initEventListeners();  // 确保 elements 已初始化
  fetchData();
}
```

### 调试技巧
1. 在 JS 文件开头添加 `console.log("🚀 文件加载")` 确认 JS 是否执行
2. 在 DOMContentLoaded 添加日志确认事件触发
3. 使用版本号强制刷新：`?v=202604011442`
