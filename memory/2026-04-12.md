## 修复 MyData API 返回数组导致成交量数据为 0 (2026-04-12 07:52 UTC)

### 用户反馈
- 显示的沪深金额不对，应该显示 2.32 万亿
- 质疑是否对接了正确的 API 接口

### 问题根因

**API 返回格式**:
```json
[
  {
    "t": "2026-04-10 00:00:00",
    "o": 3985.46,
    "h": 4011.02,
    "l": 3979.81,
    "c": 3986.23,
    "v": 546415541,
    "a": 955664107213.6,
    "pc": 3966.17,
    "sf": 0
  }
]
```

**错误代码**:
```javascript
const shData = shResp.data;  // 这是数组！
const shVolume = shData.v || 0;  // undefined → 0 ❌
const shAmountYuan = shData.a || 0;  // undefined → 0 ❌
```

**导致结果**:
- 所有成交量数据为 0
- 显示 0 亿元 instead of 23228.69 亿元

### 修复方案

#### 1. data-api.js - 修复数组处理

```javascript
// MyData API 返回的是数组，取第一个元素
const shData = Array.isArray(shResp.data) ? shResp.data[0] : shResp.data;
const szData = Array.isArray(szResp.data) ? szResp.data[0] : szResp.data;

// 使用可选链操作符
const shVolume = shData?.v || 0;
const shAmountYuan = shData?.a || 0;
```

#### 2. server.js - 非交易时间获取 API 数据

```javascript
// 非交易时间也获取一次 API 数据，用于周末复盘查看
if (isTradingTime()) {
  fetchAllData();
} else {
  console.log('⏰ 非交易时间，获取最新 API 数据...');
  fetchAllData().then(() => {
    saveCache(); // 保存最新数据到缓存
  });
}
```
