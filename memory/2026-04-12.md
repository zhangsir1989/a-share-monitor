## 二级菜单折叠按钮前置 + 自选股模块左对齐 (2026-04-12 04:44 UTC)

### 问题 1: 二级菜单折叠按钮被遮挡

**用户反馈**: 二级菜单折叠按钮被展开的二级菜单完全挡住，无法点击

**问题根因**:
```css
/* 按钮在右侧 */
.sub-sidebar-collapse-btn {
  right: -18px;  /* ❌ 在二级菜单右侧边缘 */
}
```

**问题链**:
1. 二级侧边栏展开时宽度 200px
2. 折叠按钮在右侧 (right: -18px)
3. 按钮被二级菜单内容区域覆盖
4. 用户无法点击折叠按钮 ❌

**修复方案**: 将折叠按钮移到左侧

```css
.sub-sidebar-collapse-btn {
  position: absolute;
  top: 80px;
  left: -18px;  /* ✅ 移到左侧边缘 */
  width: 36px;
  height: 36px;
  z-index: 1000;
  box-shadow: -3px 3px 10px rgba(0,0,0,0.5);  /* 阴影方向调整 */
}
```

**效果**:
- 按钮在二级菜单左侧边缘
- 主侧边栏 (240px) 和二级侧边栏 (200px) 之间有间隙
- 按钮突出到间隙中，始终可见可点击 ✅

### 问题 2: 自选股模块横向滚动条

**用户反馈**: 自选股模块在不同屏幕尺寸下，始终保持左对齐，无横向滚动条

**问题根因**:
```css
/* 强制横向滚动 */
.stock-list-section {
  overflow-x: auto;  /* ❌ 允许横向滚动 */
}

/* 强制内容宽度 */
.stock-list {
  min-width: fit-content;  /* ❌ 最小宽度=内容宽度 */
}

.stock-card {
  min-width: 600px;  /* ❌ 固定最小宽度 */
}
```

**修复方案**: 移除固定宽度限制，允许自适应

```css
/* 禁止横向滚动 */
.stock-list-section {
  overflow-x: hidden;  /* ✅ 禁止横向滚动 */
}

/* 允许缩小 */
.stock-list {
  min-width: 0;  /* ✅ 允许缩小到容器宽度 */
}

/* 允许换行 */
.stock-card {
  min-width: 0;  /* ✅ 允许缩小 */
  flex-wrap: wrap;  /* ✅ 小屏幕允许换行 */
}
```
