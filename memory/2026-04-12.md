## 隐藏主侧边栏 X 按钮 + 修复二级菜单 header 折叠按钮 (2026-04-12 05:39 UTC)

### 用户反馈
1. 主侧边栏 header 上的 X 关闭按钮要去掉
2. 二级菜单 header 上的收缩按钮移到右侧（标题旁边）
3. 二级菜单按钮点击无反应

### 问题根因

**问题 1**: 主侧边栏 header 上的 X 按钮不需要
- HTML 中存在 `<button class="btn-close-sidebar" id="close-sidebar">✕</button>`
- 需要隐藏

**问题 2**: 二级菜单 header 折叠按钮位置不对
- 按钮在 header 中但没有推到右侧
- 需要 `margin-left: auto` 推到右侧

**问题 3**: 点击无反应
- `initSubSidebarHeaderCollapse()` 中绑定了 `handleSubSidebarHeaderCollapse`
- 但该函数**未定义** ❌
- 导致点击无反应

### 修复方案

#### 1. 隐藏主侧边栏 X 按钮 (CSS)

```css
/* 隐藏主侧边栏 header 上的关闭按钮 (X) */
.btn-close-sidebar,
#close-sidebar {
  display: none !important;
}
```

#### 2. 二级菜单 header 折叠按钮移到右侧 (CSS)

```css
.sub-sidebar-header-collapse-btn {
  margin-left: auto;  /* 推到右侧 */
}
```

#### 3. 修复点击无反应 (JS)

**添加缺失的 `handleSubSidebarHeaderCollapse()` 函数**:
```javascript
function handleSubSidebarHeaderCollapse() {
  const subSidebar = document.getElementById('sub-sidebar');
  const collapseBtn = document.getElementById('sub-sidebar-header-collapse-btn');
  
  if (!subSidebar || !collapseBtn) return;
  
  sidebarState.subCollapsed = !sidebarState.subCollapsed;
  
  if (sidebarState.subCollapsed) {
    subSidebar.classList.add('collapsed');
    collapseBtn.innerHTML = '》';
  } else {
    subSidebar.classList.remove('collapsed');
    collapseBtn.innerHTML = '《';
  }
  
  updateSubSidebarButton(subSidebar);  // 同步侧面按钮
  updateLayoutClasses();
  saveSidebarState();
}
```

**添加 `updateSubSidebarHeaderButton()` 函数**:
```javascript
function updateSubSidebarHeaderButton(subSidebar) {
  const collapseBtn = document.getElementById('sub-sidebar-header-collapse-btn');
  if (!collapseBtn) return;
  
  if (subSidebar.classList.contains('collapsed')) {
    collapseBtn.innerHTML = '》';
    collapseBtn.title = '展开菜单';
  } else {
    collapseBtn.innerHTML = '《';
    collapseBtn.title = '折叠菜单';
  }
}
```

**在 `handleSubSidebarCollapse()` 中同步 header 按钮**:
```javascript
function handleSubSidebarCollapse() {
  // ... 原有逻辑
  updateSubSidebarButton(subSidebar);
  updateSubSidebarHeaderButton(subSidebar);  // ✅ 同步 header 按钮
  // ...
}
```
