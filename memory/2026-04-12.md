## 修复顶部导航栏左侧空白问题 (2026-04-12 06:29 UTC)

### 用户反馈
- 一级菜单、二级菜单全都展开情况下，顶部导航栏左侧有空白
- 希望自选股页面顶部的标题栏和下方的搜索、行情模块左对齐

### 问题根因

**双重 padding 问题**:
- `.container` 有 `padding: 20px`（通用内边距）
- `.header` 也有 `padding: 20px`（自身内边距）
- 导致 header 左侧有**双重 padding**：20px + 20px = 40px 空白 ❌

```
container padding-left: 460px (侧边栏空间) + 20px (通用 padding)
                              ↓
header padding: 20px (自身 padding)
                              ↓
实际左侧空白：20px + 20px = 40px ❌
```

### 修复方案

```css
/* 修改前 */
.header {
  padding: 20px;  /* 上下左右都是 20px */
}

/* 修改后 */
.header {
  padding: 20px;
  padding-left: 0;  /* 移除左侧 padding，与下方模块对齐 */
}
```

**效果**:
- header 左侧只保留 container 的 20px padding
- 与下方模块（`.add-stock-section` 等）的 `padding-left: 0` 保持一致
- 所有模块左边界垂直对齐 ✅

### 文件修改
