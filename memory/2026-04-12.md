## 统一两个折叠按钮样式 + 增加垂直间距 (2026-04-12 04:54 UTC)

### 用户反馈
- 两个折叠按钮间距太近，重叠在一起
- 形状大小不一致
- 图标需要前置

### 优化方案

#### 1. 统一按钮尺寸和样式

| 属性 | 修改前（主按钮） | 修改后 |
|------|-----------------|--------|
| 尺寸 | 28px × 28px | 36px × 36px ✅ |
| 字体 | 14px | 20px ✅ |
| 背景色 | var(--bg-card) | #2f363d ✅ |
| 边框 | 1px | 2px ✅ |
| 阴影 | 2px 2px 8px | -3px 3px 10px ✅ |

#### 2. 统一按钮位置

```css
/* 主侧边栏按钮 */
.sidebar-collapse-btn {
  left: -18px;  /* 从 right 移到左侧 */
  top: 80px;
}

/* 二级侧边栏按钮 */
.sub-sidebar-collapse-btn {
  left: -18px;  /* 保持左侧 */
  top: 130px;   /* 向下移动 50px */
}
```

#### 3. 增加垂直间距

- **主侧边栏按钮**: top: 80px
- **二级侧边栏按钮**: top: 130px
- **垂直间距**: 50px ✅（不再重叠）

#### 4. 统一交互效果

```css
/* 悬停效果 */
:hover {
  background: #1890ff;  /* 蓝色 */
  transform: scale(1.15);
}
```
