## 一级菜单折叠按钮上移与二级菜单按钮同一行 (2026-04-12 05:57 UTC)

### 用户反馈
- 一级菜单折叠按钮往上移动
- 和二级菜单折叠菜单按钮在同一行

### 修复方案

#### 1. 一级菜单折叠按钮位置

```css
/* 修改前 */
.sidebar-collapse-btn {
  top: 50%;
  transform: translateY(-50%);  /* 垂直居中 */
}

/* 修改后 */
.sidebar-collapse-btn {
  top: 20px;  /* 与 header 顶部对齐 */
  /* 移除 transform */
}
```

#### 2. 二级菜单 header 折叠按钮位置

```css
/* 修改前 */
.sub-sidebar-header-collapse-btn {
  /* flex 布局，在 header 内部 */
  margin-left: auto;
}

/* 修改后 */
.sub-sidebar-header-collapse-btn {
  position: absolute;
  top: 20px;  /* 与一级菜单按钮同一行 */
  right: 15px;  /* 在二级侧边栏内部右侧 */
}
```

#### 3. 对齐逻辑

- **一级菜单 header** 高度约 60-70px，按钮在 `top: 20px`
- **二级菜单 header** padding: `20px 15px`，按钮在 `top: 20px`
- **两个按钮** 都在 `top: 20px`，同一水平线 ✅
