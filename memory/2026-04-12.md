## 调整折叠按钮位置到用户指定位置 (2026-04-12 05:03 UTC)

### 用户反馈
- 主侧边栏折叠按钮应该在主侧边栏右侧边缘
- 二级侧边栏折叠按钮应该在二级侧边栏左侧边缘
- 即使收缩了也要都能展示出来

### 修复方案

#### 1. 主侧边栏折叠按钮

```css
.sidebar-collapse-btn {
  position: absolute;
  top: 50%;  /* 垂直居中 */
  transform: translateY(-50%);  /* 精确居中 */
  right: -18px;  /* 主侧边栏右侧边缘 */
  box-shadow: 3px 3px 10px rgba(0,0,0,0.5);  /* 右侧阴影 */
}
```

#### 2. 二级侧边栏折叠按钮

```css
.sub-sidebar-collapse-btn {
  position: absolute;
  top: 50%;  /* 垂直居中 */
  transform: translateY(-50%);  /* 精确居中 */
  left: -18px;  /* 二级侧边栏左侧边缘 */
  box-shadow: -3px 3px 10px rgba(0,0,0,0.5);  /* 左侧阴影 */
}
```

#### 3. 折叠状态保持

```css
/* 主侧边栏折叠时 */
.sidebar.collapsed .sidebar-collapse-btn {
  right: -18px;  /* 保持在右侧 */
}

/* 二级侧边栏折叠时 */
.sub-sidebar.collapsed .sub-sidebar-collapse-btn {
  left: -18px;  /* 保持在左侧 */
}
```
