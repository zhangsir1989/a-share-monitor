## 统一两个折叠按钮样式大小 (2026-04-12 06:39 UTC)

### 用户反馈
- 一级菜单和二级菜单的两个折叠按钮大小不一致
- 需要统一样式

### 问题根因

**样式不一致**:
| 属性 | 一级菜单按钮 | 二级菜单按钮 |
|------|-------------|-------------|
| font-size | 20px | 18px ❌ |
| color | #f0f6fc | var(--text-secondary) ❌ |
| hover color | #1890ff | var(--accent-color) ❌ |
| hover background | rgba(255,255,255,0.1) | var(--bg-primary) ❌ |

### 修复方案

```css
/* 二级菜单按钮修改后 - 与一级菜单完全一致 */
.sub-sidebar-header-collapse-btn {
  font-size: 20px;  /* 18px → 20px */
  color: #f0f6fc;  /* var(--text-secondary) → #f0f6fc */
  font-weight: bold;
  padding: 4px 8px;
  border-radius: 4px;
}

.sub-sidebar-header-collapse-btn:hover {
  color: #1890ff;  /* var(--accent-color) → #1890ff */
  background: rgba(255, 255, 255, 0.1);  /* var(--bg-primary) → rgba(255,255,255,0.1) */
}
```
