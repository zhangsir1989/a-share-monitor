## 修复二级菜单 header 折叠按钮未显示 (2026-04-12 05:29 UTC)

### 用户反馈
二级菜单 header 上没有显示折叠按钮（红圈位置空白）

### 问题根因

1. **JS 问题**: `initSubSidebarHeaderCollapse()` 函数定义了，但在 `initSidebar()` 中没有被调用
   ```javascript
   // ❌ 修改前
   if (subSidebar) {
     initSubSidebarCollapse();
   }
   // 缺少 initSubSidebarHeaderCollapse() 调用
   ```

2. **CSS 问题**: `.sub-sidebar-header` 缺少 flex 布局，按钮样式不完整
   ```css
   /* ❌ 修改前 */
   .sub-sidebar-header {
     padding: 20px 15px;
     border-bottom: 1px solid var(--border-color);
     position: relative;
   }
   /* 缺少 display: flex 和按钮样式 */
   ```

### 修复方案

#### 1. JS 修复 (sidebar.js)

```javascript
// ✅ 修改后
if (subSidebar) {
  initSubSidebarCollapse();
  initSubSidebarHeaderCollapse();  // ✅ 添加调用
}
```

#### 2. CSS 修复 (sidebar.css)

```css
/* ✅ 修改后 */
.sub-sidebar-header {
  padding: 20px 15px;
  border-bottom: 1px solid var(--border-color);
  display: flex;              /* ✅ 添加 flex 布局 */
  align-items: center;        /* ✅ 垂直居中 */
  justify-content: space-between;  /* ✅ 两端对齐 */
  position: relative;
}

/* ✅ 添加按钮样式 */
.sub-sidebar-header-collapse-btn {
  background: transparent;
  border: none;
  cursor: pointer;
  color: var(--text-secondary);
  font-size: 18px;
  font-weight: bold;
  padding: 4px 8px;
  border-radius: 4px;
  transition: all 0.2s;
  display: flex;
  align-items: center;
  justify-content: center;
  flex-shrink: 0;  /* ✅ 防止被压缩 */
}

.sub-sidebar-header-collapse-btn:hover {
  background: var(--bg-primary);
  color: var(--accent-color);
}
```
