**updatePagination 函数修复**：
```javascript
function updatePagination(tableType, currentPage, totalPages) {
  // 转换驼峰命名为连字符命名（匹配 HTML 中的 ID）
  let idPrefix;
  if (tableType === 'limitUpStocks') {
    idPrefix = 'limit-up';
  } else if (tableType === 'limitDownStocks') {
    idPrefix = 'limit-down';
  } else if (tableType === 'strongStocks') {
    idPrefix = 'strong';
  }
  
  const currentEl = document.getElementById(`${idPrefix}-current-page`);
  const totalEl = document.getElementById(`${idPrefix}-total-pages`);
  const prevBtn = document.getElementById(`${idPrefix}-prev`);
  const nextBtn = document.getElementById(`${idPrefix}-next`);
  
  // 更新显示
  if (currentEl) currentEl.textContent = currentPage;
  if (totalEl) totalEl.textContent = totalPages;
  
  // 更新按钮状态
  if (prevBtn) prevBtn.disabled = currentPage === 1;
  if (nextBtn) nextBtn.disabled = currentPage === totalPages;
  
  console.log(`分页更新：${tableType} 当前页 ${currentPage} / 总页数 ${totalPages}`);
}
```

function getLatestTradeDate() {
  const today = new Date();
  const weekday = today.getDay();
  let tradingDate = new Date(today);
  
  if (weekday === 0) { // 周日
    tradingDate.setDate(today.getDate() - 2); // 回退到周五
  } else if (weekday === 6) { // 周六
    tradingDate.setDate(today.getDate() - 1); // 回退到周五
  }
  // 周一至周五直接使用当天
  
  return tradingDate.toISOString().split('T')[0];
}
```

| 2026-04-09 | 周四 | 2026-04-09 | 0 天 |

### 用户反馈

**错误日志**：
```

### 问题根因 🔍

**原因**：
- `initDatePickers()` 函数使用 `new Date()` 获取今天日期
- 2026-04-12 是**周日**，非交易日
- MyData API 对非交易日返回 **404 错误**
- 导致跌停个股等数据获取失败，只显示 3 只错误数据

### 修复方案 ✅

**新增函数**（public/js/app.js）：
```javascript
function getLatestTradeDate() {
  const today = new Date();
  const weekday = today.getDay();
  let tradingDate = new Date(today);
  if (weekday === 0) { // 周日
    tradingDate.setDate(today.getDate() - 2); // 回退到周五
  } else if (weekday === 6) { // 周六
    tradingDate.setDate(today.getDate() - 1); // 回退到周五
  }
  return tradingDate.toISOString().split('T')[0];
}
```

**修改**：
- `initDatePickers()` 从使用 `new Date()` 改为使用 `getLatestTradeDate()`
