## 优化主侧边栏展开时自动展开二级侧边栏 (2026-04-12 04:29 UTC)

### 用户反馈
点击主侧边栏的》点击箭头展开后，二级菜单没有正常显示

### 问题根因
**场景**: 用户折叠了主侧边栏和二级侧边栏 → 点击主侧边栏的》按钮展开

**问题链**:
1. 主侧边栏折叠，二级侧边栏也折叠
2. 用户点击主侧边栏的》按钮展开主侧边栏
3. `handleMainSidebarCollapse()` 只处理主侧边栏状态
4. 二级侧边栏保持折叠状态 ❌
5. 用户看不到二级菜单选项 ❌

### 优化方案

**在 `handleMainSidebarCollapse()` 函数中添加联动逻辑**:

```javascript
function handleMainSidebarCollapse() {
  const sidebar = document.getElementById('sidebar');
  const subSidebar = document.getElementById('sub-sidebar');
  const collapseBtn = document.getElementById('sidebar-collapse-btn');
  const subCollapseBtn = document.getElementById('sub-sidebar-collapse-btn');
  
  sidebarState.mainCollapsed = !sidebarState.mainCollapsed;
  
  if (sidebarState.mainCollapsed) {
    // 折叠主侧边栏
    sidebar.classList.add('collapsed');
    collapseBtn.innerHTML = '》';
    console.log('✅ 主侧边栏折叠');
  } else {
    // 展开主侧边栏
    sidebar.classList.remove('collapsed');
    collapseBtn.innerHTML = '《';
    
    // ✅ 优化：如果当前页面有二级侧边栏，自动展开
    if (subSidebar) {
      subSidebar.classList.remove('collapsed');
      sidebarState.subCollapsed = false;
      if (subCollapseBtn) {
        subCollapseBtn.innerHTML = '《';
      }
      console.log('✅ 主侧边栏展开，同时展开二级侧边栏');
    } else {
      console.log('✅ 主侧边栏展开');
    }
  }
  
  updateLayoutClasses();
  saveSidebarState();
}
```
