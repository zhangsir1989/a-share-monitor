## 2026-04-13 11:45 UTC - 分组管理功能重构完成（type 字段一对一关联）

### 用户需求
- 新建分组时生成新的 type 值
- 股票的 type 字段 = 所属分组的 type
- 一只股票只能属于一个分组
- 默认自选股（type=1）独立，不显示其他分组的股票
- 系统重启后正常工作

### 核心设计

**一对一关系**：
```
custom_groups 表：每个分组有唯一 type 值
    ↓
custom_stocks 表：type 字段 = 股票所属分组的 type
    ↓
一只股票只能属于一个分组（通过 type 字段关联）
```

**type 值分配**：
- `type=1`：默认自选股（系统内置，不可删除）
- `type=2,3,4...`：用户创建的分组（自动递增）

### 数据库表结构

#### custom_groups 表
```sql
CREATE TABLE custom_groups (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id TEXT NOT NULL,
  name TEXT NOT NULL,
  type INTEGER NOT NULL UNIQUE,  -- 唯一 type 值
  icon TEXT DEFAULT '📁',
  color TEXT DEFAULT '#4a9eff',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

#### custom_stocks 表
```sql
CREATE TABLE custom_stocks (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id TEXT NOT NULL,
  stock_code TEXT NOT NULL,
  stock_market TEXT NOT NULL,
  type INTEGER DEFAULT 1,  -- 关联 custom_groups.type
  added_at DATETIME DEFAULT CURRENT_TIMESTAMP
  -- 移除了 UNIQUE(user_id, stock_code, stock_market, type) 约束
);
```

### 后端 API（group-api.js）

| 端点 | 方法 | 说明 | 参数 | 返回 |
|------|------|------|------|------|
| `/api/custom-groups/list` | GET | 获取分组列表（含股票数量） | 无 | `{groups: [{id, name, type, icon, color, stockCount}], stockGroups: {code: type}}` |
| `/api/custom-groups/create` | POST | 创建分组（自动分配 type） | `{name, icon, color}` | `{id, name, type, icon, color}` |
| `/api/custom-groups/update` | POST | 更新分组 | `{id, name, icon, color}` | `{success: true}` |
| `/api/custom-groups/delete` | POST | 删除分组（股票回归 type=1） | `{id}` | `{success: true}` |
| `/api/custom-groups/assign` | POST | 分配股票到分组 | `{code, market, type}` | `{success: true}` |
| `/api/custom-groups/remove` | POST | 从分组移除（回归 type=1） | `{code, market}` | `{success: true}` |

### 前端逻辑（group-manager.js）

**状态结构**：
```javascript
const GroupManager = {
  groups: [],  // [{id, name, type, icon, color, stockCount}]
```

**核心函数**：
- `assignStockToGroup(code, type)` - 分配股票到分组（单选）
- `removeStockFromGroup(code)` - 从分组移除（回归 type=1）
- `renderGroupSelector(code)` - 显示当前分组，点击切换
- `getFilteredStocks()` - 按 type 筛选股票

### 迁移逻辑（server.js）

```javascript
// 如果 custom_groups 表没有 type 字段，添加它
try {
  db.exec(`ALTER TABLE custom_groups ADD COLUMN type INTEGER`);
  console.log('✅ 分组表已添加 type 字段');
  
  // 为现有分组分配 type 值（从 2 开始）
  const existingGroups = db.exec(`SELECT id FROM custom_groups WHERE type IS NULL`);
  if (existingGroups.length > 0) {
    let typeCounter = 2;
    existingGroups[0].values.forEach(row => {
      const id = row[0];
      db.run(`UPDATE custom_groups SET type = ${typeCounter} WHERE id = ${id}`);
      typeCounter++;
    });
    console.log(`✅ 已为 ${existingGroups[0].values.length} 个现有分组分配 type 值`);
  }
} catch (e) {
  // 列已存在，忽略
}
```
