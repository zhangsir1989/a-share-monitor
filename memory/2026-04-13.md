## 2026-04-13 15:10 UTC - 修复分组切换后股票列表不显示的问题

### 问题描述

**用户反馈**：
- AI 算力分组标签显示有 1 个股票
- 但点击"AI 算力"标签后，页面显示"暂无自选股"
- 后端 API 正常返回数据

**截图证据**：
- 分组标签："AI 算力 1"（显示有 1 个股票）
- 页面内容："暂无自选股"（空列表）

### 问题分析

#### 1. 后端 API 正常

```bash
curl "http://localhost:3000/api/custom-stocks/list?type=2"
# 返回：
{
  "success": true,
  "data": [
    {"code": "603019", "market": "sh", "addedAt": "2026-04-13 15:02:08"}
  ]
}
```

✅ 后端正常返回 AI 算力分组（type=2）的股票数据

#### 2. 前端状态变量不同步

**GroupManager 状态**：
```javascript
const GroupManager = {
  currentFilter: 1,  // 当前筛选的分组 type（用于显示股票）
  currentType: 1,    // 当前选中的分组 type（用于添加股票）
  // ...
};
```

**问题流程**：
```
1. 用户点击"AI 算力"标签 (type=2)
   ↓
2. switchGroup(2) 被调用
   ↓
3. GroupManager.currentType = 2  ✅ 更新
4. GroupManager.currentFilter = 1  ❌ 未更新（仍是默认值）
   ↓
5. loadStocks(2) 从后端加载 type=2 的股票
   ↓
6. pageState.stocks = [603019]  ✅ 加载成功
   ↓
7. updateStockList() 调用 getFilteredStocks()
   ↓
8. getFilteredStocks() 检查 currentFilter === 1
   ↓
9. 筛选条件：type === 1  ❌ 但股票 type=2
   ↓
10. filteredStocks = []  ❌ 空数组
   ↓
11. 页面显示"暂无自选股"
```

#### 3. 核心问题

**getFilteredStocks() 函数**：
```javascript
function getFilteredStocks() {
  if (GroupManager.currentFilter === 1 || GroupManager.currentFilter === 'all') {
    // 显示默认自选股（type=1）
    return pageState.stocks.filter(stock => {
      const type = GroupManager.stockGroups[stock.code] || 1;
      return type === 1;  // ❌ 筛选 type=1 的股票
    });
  }
  
  // 显示指定分组的股票
  return pageState.stocks.filter(stock => {
    const type = GroupManager.stockGroups[stock.code] || 1;
    return type === GroupManager.currentFilter;  // ❌ 使用 currentFilter
  });
}
```

**switchGroup() 函数（修复前）**：
```javascript
async function switchGroup(type) {
  GroupManager.currentType = type;  // ✅ 更新
  // ❌ 未更新 currentFilter
  await loadStocks(type);
}
```

**结果**：
- `pageState.stocks` 包含 type=2 的股票
- `currentFilter` 仍是 1
- `getFilteredStocks()` 筛选 type=1 的股票
- 结果：空数组

### 修复方案

#### 修复 1: switchGroup() 同步更新两个变量

**修改前**：
```javascript
async function switchGroup(type) {
  if (GroupManager.currentType === type) {
    return;
  }
  
  GroupManager.currentType = type;  // ✅
  // ❌ 缺少：GroupManager.currentFilter = type;
  
  renderGroupTabs();
  await loadStocks(type);
}
```

**修改后**：
```javascript
async function switchGroup(type) {
  if (GroupManager.currentType === type) {
    return;
  }
  
  GroupManager.currentType = type;      // ✅ 用于添加股票
  GroupManager.currentFilter = type;    // ✅ 用于筛选股票（新增）
  
  renderGroupTabs();
  await loadStocks(type);
  
  console.log('📁 切换到分组:', groupName, 
    '(type=' + type + ', currentFilter=' + GroupManager.currentFilter + ')');
}
```

#### 修复 2: filterByGroup() 同步更新两个变量

**修改前**：
```javascript
async function filterByGroup(newType) {
  GroupManager.currentFilter = newType;  // ✅
  // ❌ 缺少：GroupManager.currentType = newType;
  
  await loadStocks(newType);
}
```

**修改后**：
```javascript
async function filterByGroup(newType) {
  GroupManager.currentFilter = newType;  // ✅ 用于筛选股票
  GroupManager.currentType = newType;    // ✅ 用于添加股票（新增）
  
  await loadStocks(newType);
}
```

