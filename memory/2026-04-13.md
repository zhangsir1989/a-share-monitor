## 个股详细页数据不显示修复 (2026-04-13 05:05 UTC)

**用户反馈**：个股详细页没数据，所有字段显示"--"

**问题根因**：
1. `loadBasicInfo()` 函数只调用 `UI.updateBasicInfo()` 更新头部（名称、价格、涨幅）
2. 没有调用 `UI.updateBasicData()` 更新基本数据表格
3. 导致基本数据表格所有字段显示"--"
4. 代码引用错误：`Chart.drawPlaceholder` 应为 `IntradayChart.drawPlaceholder`

**修复内容**：
1. **loadBasicInfo() 函数修复**：
   ```javascript
   // 修复前：
   UI.updateBasicInfo(result.data);  // 只更新头部
   
   // 修复后：
   UI.updateBasicInfo(result.data);  // 更新头部
   UI.updateBasicData(result.data);  // 更新基本数据表格
   ```

2. **错误引用修复**：
   ```javascript
   // 修复前：
   Chart.drawPlaceholder('分时数据加载失败');
   
   // 修复后：
   IntradayChart.drawPlaceholder('分时数据加载失败');
   ```

**文件修改**：
1. `public/stock-detail/js/main.js`:
   - 修复 `loadBasicInfo()` 函数，添加 `UI.updateBasicData()` 调用
   - 修复 `Chart.drawPlaceholder` → `IntradayChart.drawPlaceholder`

2. `public/stock-detail/index.html`:
   - 所有 JS 文件版本号更新为 v=2026041304

3. 删除未使用文件：
   - `public/stock-detail/js/chart.js`（未被引用，526 行）
