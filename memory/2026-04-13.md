## 2026-04-13 14:35 UTC - 分组下拉菜单居中显示并添加遮罩层

### 用户需求

**分组管理下拉菜单在页面正中间显示**

从截图看，用户点击"分组管理"按钮后，下拉菜单显示在按钮右下角，用户希望它在页面正中间显示，像一个对话框。

### 问题分析

**原布局**：
```css
.group-dropdown {
  position: absolute;
  top: 100%;    /* 按钮下方 */
  right: 0;     /* 右对齐 */
}
```

**问题**：下拉菜单跟随按钮位置，在页面右下角显示

**目标**：下拉菜单在页面正中间显示，像对话框一样

### 实现方案

#### 1. 修改下拉菜单定位

**修改前**：
```css
.group-dropdown {
  position: absolute;
  top: 100%;
  right: 0;
  min-width: 320px;
  max-width: 400px;
  max-height: 400px;
  border-radius: 8px;
  z-index: 1000;
}
```

**修改后**：
```css
.group-dropdown {
  position: fixed;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);  /* 精确居中 */
  min-width: 360px;
  max-width: 420px;
  max-height: 500px;
  border-radius: 12px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.5);
  z-index: 2000;
}
```

**关键改动**：
- `position: fixed` - 相对于视口定位
- `top: 50%; left: 50%` - 左上角在中心
- `transform: translate(-50%, -50%)` - 向左上偏移自身宽高的一半，实现精确居中
- 增大尺寸和圆角，更像对话框
- 增强阴影，提高视觉层次

#### 2. 修改动画效果

**修改前**（slideDown）：
```css
@keyframes slideDown {
  from { transform: translateY(-10px); }
  to { transform: translateY(0); }
}
```

**修改后**（zoomIn）：
```css
@keyframes zoomIn {
  from {
    opacity: 0;
    transform: translate(-50%, -50%) scale(0.9);
  }
  to {
    opacity: 1;
    transform: translate(-50%, -50%) scale(1);
  }
}
```

**效果**：从中心点缩放淡入，更符合对话框的感觉

#### 3. 添加遮罩层

**HTML**：
```html
<div id="group-overlay" class="group-overlay" style="display: none;"></div>
```

**CSS**：
```css
.group-overlay {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0, 0, 0, 0.6);
  z-index: 1999;
  display: none;
}

.group-overlay.show {
  display: block;
}
```

**作用**：
- 半透明黑色背景，突出中间的对话框
- 点击遮罩层关闭对话框
- 视觉层次：遮罩层 (1999) < 下拉菜单 (2000)

#### 4. 修改 JS 逻辑

**修改前**：
```javascript
GroupManager.elements.groupManagerBtn?.addEventListener('click', (e) => {
  e.stopPropagation();
  GroupManager.elements.groupDropdown.classList.toggle('show');
});

document.addEventListener('click', () => {
  GroupManager.elements.groupDropdown?.classList.remove('show');
});
```

**修改后**：
```javascript
GroupManager.elements.groupManagerBtn?.addEventListener('click', (e) => {
  e.stopPropagation();
  const dropdown = GroupManager.elements.groupDropdown;
  const overlay = document.getElementById('group-overlay');
  
  if (dropdown.classList.contains('show')) {
    // 关闭
    dropdown.classList.remove('show');
    overlay?.classList.remove('show');
  } else {
    // 打开
    dropdown.classList.add('show');
    overlay?.classList.add('show');
  }
});

// 点击遮罩层关闭
document.getElementById('group-overlay')?.addEventListener('click', () => {
  GroupManager.elements.groupDropdown?.classList.remove('show');
  document.getElementById('group-overlay')?.classList.remove('show');
});

// 点击其他地方关闭
document.addEventListener('click', () => {
  GroupManager.elements.groupDropdown?.classList.remove('show');
  document.getElementById('group-overlay')?.classList.remove('show');
});
```
