## 分时图数据不显示修复 (2026-04-13 05:32 UTC)

**用户反馈**：分时没数据显示，五档买卖请自动刷新

**问题根因**：
- `API.getIntraday()` 返回 `{ success: true, data: result }`
- 但 `result` 已经是完整响应对象 `{ success, prevClose, data }`
- 导致 `IntradayChart.render()` 收到错误的数据结构
- 应该返回 `result.data`（数据数组）而不是 `result`（整个响应）

**修复内容**：

1. **API.getIntraday() 返回格式修复**：
   ```javascript
   // 修复前（错误）：
   return { success: true, data: result };
   // result = { success: true, prevClose: 11.38, data: [...] }
   
   // 修复后（正确）：
   return { 
     success: true, 
     data: result.data,      // 数据数组
     prevClose: result.prevClose  // 昨收价
   };
   ```

2. **五档买卖自动刷新**：
   - 已在 `startAutoRefresh()` 中配置 500ms 刷新间隔
   - 交易时间自动刷新（9:15-11:30, 13:00-15:00）
   - 刷新函数：`loadOrderBook()`

**文件修改**：
1. `public/stock-detail/js/api.js`:
   - 修复 `getIntraday()` 返回格式
   - 返回 `result.data` 和 `prevClose`

2. `public/stock-detail/index.html`:
   - api.js 版本号更新：`v=2026041305`
