## 2026-04-13 13:55 UTC - 数据管理新增分组查询功能

### 用户需求

1. 新增分组表 `custom_groups`，字段：分组类型 (type)、分组名称 (name)
2. 分组类型从 2 开始编号，默认分组"自选股"类型为 1
3. 新增分组时插入该表
4. 分组管理支持删除分组
5. 数据管理新增二级菜单"分组查询"，功能：查询分组表

### 数据库表结构 (custom_groups)

```sql
CREATE TABLE custom_groups (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id TEXT NOT NULL,
  name TEXT NOT NULL,
  icon TEXT DEFAULT '📁',
  color TEXT DEFAULT '#4a9eff',
  type INTEGER NOT NULL UNIQUE,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

**字段说明**：
- `type`: 分组类型，UNIQUE 约束
  - `type=1`: 默认自选股（系统预置）
  - `type=2+`: 用户创建的分组（自动递增）

### 分组管理功能

#### 创建分组
```javascript
// POST /api/custom-groups/create
{
  "name": "测试分组",
  "icon": "🧪",
  "color": "#ff6b6b"
}

// 返回
{
  "success": true,
  "data": {
    "type": 4  // 自动分配，从 2 开始递增
  }
}
```

#### 删除分组
```javascript
// POST /api/custom-groups/delete
{
  "id": 5
}

// 行为：
// 1. 将该分组的股票 type 更新为 1（回归默认自选股）
// 2. 删除 custom_groups 记录
// 3. 保存数据库
```
