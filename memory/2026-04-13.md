### 核心问题与解决

#### 问题：saveDatabase 闭包失效
**现象**：group-api.js 中调用 saveDatabase() 后，数据库没有保存
**原因**：saveDatabase 函数在 server.js 中定义，引用 DB_PATH 变量。当传递给 group-api.js 模块后，由于模块加载机制，闭包可能未正确捕获 DB_PATH
**解决**：在 group-api.js 模块内部定义 saveDatabase 函数，直接导入 DB_PATH

```javascript
// group-api.js
const fs = require('fs');
const path = require('path');
const DB_PATH = path.join(__dirname, '../data/users.db');

function saveDatabase(db) {
  if (db) {
    const data = db.export();
    const buffer = Buffer.from(data);
    fs.writeFileSync(DB_PATH, buffer);
  }
}

module.exports = function(app, db) {
  // 使用 saveDatabase(db)
};
```
