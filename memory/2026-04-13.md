## 2026-04-13 15:30 UTC - 修复分组标签计数错误和添加股票后不显示的问题

### 用户反馈

**问题 1**: AI 算力分组有股票但标签显示 0
- 截图显示："AI 算力 0"（红圈标记）
- 但实际后端有 2 只股票 (603019, 000977)

**问题 2**: 添加股票后需要刷新页面才能看到
- 在分组中添加证券
- 页面不立即显示，需要刷新

### 问题分析

#### 问题 1: 分组标签计数错误

**后端数据正确**：
```json
{
  "groups": [
    {"type": 1, "name": "我的自选股", "stockCount": 0},
    {"type": 2, "name": "AI 算力", "stockCount": 2},
    {"type": 3, "name": "机器人", "stockCount": 2}
  ],
  "stockGroups": {
    "159530": 3,
    "399283": 3,
    "603019": 2,
    "000977": 2
  }
}
```

**前端显示错误**：
- 我的自选股：1 ❌（应为 0）
- AI 算力：0 ❌（应为 2）
- 机器人：1 ❌（应为 2）

**根本原因**：
```javascript
// ❌ 错误的计数逻辑
const stockCounts = {};
GroupManager.groups.forEach(g => {
  // pageState.stocks 只包含当前筛选分组的股票！
  stockCounts[g.type] = pageState.stocks.filter(s => 
    (GroupManager.stockGroups[s.code] || 1) === g.type
  ).length;
});
```

**问题流程**：
```
1. 当前在"机器人"分组 (type=3)
2. pageState.stocks = [159530, 399283] (只有 type=3 的股票)
3. 计算 AI 算力 (type=2) 的数量：
   - filter 检查 pageState.stocks 中 type=2 的股票
   - pageState.stocks 只有 type=3 的股票
   - 结果：0 ❌
4. 计算我的自选股 (type=1) 的数量：
   - filter 检查 pageState.stocks 中 type=1 的股票
   - 但 159530 和 399283 的 stockGroups 值是 3
   - stockGroups[code] || 1 → 3 || 1 → 3
   - 3 !== 1，被过滤
   - 结果：0 ❌
```

**正确做法**：
- 直接使用 `GroupManager.stockGroups` 统计
- 它包含所有股票的分组映射，不依赖 pageState.stocks

#### 问题 2: 添加股票后不显示

**问题流程**：
```
1. 当前在"AI 算力"分组 (currentType=2, currentFilter=2)
2. pageState.stocks = [603019, 000977]
3. GroupManager.stockGroups = {603019: 2, 000977: 2, ...}
4. 用户添加新股票 300001
5. saveStockToDb(code, market, type=2)  ✅
6. pageState.stocks.push(300001)  ✅
7. GroupManager.stockGroups 没有更新  ❌
8. updateStockList() → getFilteredStocks()
9. getFilteredStocks() 检查 stockGroups[300001] || 1 → 1
10. 1 !== 2 (currentFilter)，300001 被过滤  ❌
11. 页面不显示新股票
```

**根本原因**：
- `addStock()` 后 `GroupManager.stockGroups` 没有更新
- `getFilteredStocks()` 使用错误的映射值过滤新股票

### 修复方案

#### 修复 1: 改用 stockGroups 统计分组数量

**renderGroupTabs() 修改前**：
```javascript
const stockCounts = {};
GroupManager.groups.forEach(g => {
  stockCounts[g.type] = pageState.stocks.filter(s => 
    (GroupManager.stockGroups[s.code] || 1) === g.type
  ).length;  // ❌ 依赖 pageState.stocks
});
```

**修改后**：
```javascript
const stockCounts = {};
GroupManager.groups.forEach(g => {
  stockCounts[g.type] = 0;  // 初始化为 0
});

// 遍历 stockGroups 统计每个分组的股票数
Object.values(GroupManager.stockGroups || {}).forEach(type => {
  if (stockCounts[type] !== undefined) {
    stockCounts[type]++;  // ✅ 直接统计
  }
});
```

**updateGroupTabCounts() 同样修改**：
```javascript
function updateGroupTabCounts(stocks = []) {
  const stockCounts = {};
  GroupManager.groups.forEach(g => {
    stockCounts[g.type] = 0;
  });
  
  Object.values(GroupManager.stockGroups || {}).forEach(type => {
    if (stockCounts[type] !== undefined) {
      stockCounts[type]++;
    }
  });
  
  // 更新 DOM
  document.querySelectorAll('.group-tab').forEach(tab => {
    const type = parseInt(tab.dataset.type);
    const countEl = tab.querySelector('.stock-count');
    if (countEl) {
      countEl.textContent = stockCounts[type] || 0;
    }
  });
}
```

#### 修复 2: 添加股票后更新 stockGroups

**addStock() 修改**：
```javascript
async function addStock(code, name, market) {
  // ... 保存数据库
  
  // 添加到本地列表
  pageState.stocks.push({code, market, name, addedAt: Date.now()});
  
  // ✅ 新增：更新分组映射
  const currentType = GroupManager.currentType || 1;
  if (typeof GroupManager !== 'undefined') {
    GroupManager.stockGroups[code] = currentType;
    console.log('📦 更新分组映射:', code, '→ type', currentType);
  }
  
  updateStockList();
  fetchAllStockData();
  
  // ✅ 新增：更新分组标签计数
  if (typeof updateGroupTabCounts === 'function') {
    updateGroupTabCounts();
  }
  
  showToast(`已添加 ${name}`, 'success');
}
```

#### 修复 3: 删除股票后更新 stockGroups

**removeStock() 修改**：
```javascript
async function removeStock(code) {
  // ... 确认删除
  
  if (success) {
    const index = pageState.stocks.findIndex(s => s.code === code);

      updateStockList();

  }
```
