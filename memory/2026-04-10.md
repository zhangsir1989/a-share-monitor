# 2026-04-10 - A 股实时监控服务开发记录

## 分时图功能优化

### MyData API 对接
- 使用接口：`https://api.mairuiapi.com/hsstock/history/{code}.{market}/5/n/{licence}?st={date}&et={date}&lt=48`
- 获取 5 分钟 K 线数据（48 条），转换为分时图格式
- 后续改为使用 `hsstock/latest` 接口获取实时数据

### 数据库存储
- 表名：`intraday_data`
- 字段：stock_code, trade_date, time, price, open, high, low, volume, amount, prevClose
- 逻辑：历史数据从数据库获取，当前时间数据从 API 获取并存储

### currentIndex 计算修复
- 上午交易时间（9:30-11:30）：索引 0-120
- 午休时间（11:30-13:00）：停留在索引 120
- 下午交易时间（13:00-15:00）：索引 121-241
- 价格曲线只绘制到当前时间

## 分时数据查询功能

### 页面路径
- `/intraday-data.html`
- 数据管理 → 分时数据查询（二级菜单）

### 查询条件
- 股票代码：6 位代码精确匹配
- 交易日期：日期选择器，默认今天
- 市场：全部/沪市/深市/港股

### 显示字段
- 时间、价格、开盘、最高、最低、成交量、成交额、昨收、涨跌
- 分页：50 条/页
- 涨跌颜色：红涨绿跌

### 后端 API
- `GET /api/intraday-data`
- 查询 intraday_data 表
- 支持多条件组合筛选
